# 商家资料操作手册

## 功能定位

商家资料管理用于平台/厂家租户查看所有商家资料，并维护商家的联系资料、地址资料和备注。

## 页面入口

- 菜单：商家管理 / 商家资料
- 路由：`/merchantManagement/merchantProfile`
- 兼容路由：`/system/merchant/profile`

## 常用操作

1. 按商家名称或主邮箱查询。
2. 查看商家详情。
3. 编辑联系人、电话、地址和备注。

当前普通编辑不维护商家名称、公司名称、主邮箱、国家或审核状态。

## AI 代办能力

### 支持的意图

```text
意图名称：平台更新商家联系资料
用户说法：把 ABC Dealer 的联系人手机号改成 13800000000。
```

### 必填槽位

- 目标商家。
- 至少一个可编辑字段。

### 需要追问

- 目标商家无法唯一定位。
- 用户要求修改主体锁定字段。

### 保存前确认摘要

```text
将更新商家资料：
- 商家：ABC Dealer
- 手机：13800000000
是否确认保存？
```
