# 商家资料接口与数据契约

## 页面与权限

| 项目 | 内容 |
| --- | --- |
| 菜单 | 商家管理 / 商家资料 |
| 前端路由 | `/merchantManagement/merchantProfile` |
| 前端组件 | `admin-ui/src/pages/system/MerchantProfilePage.vue` |
| 权限码 | `system:merchant:profile:*` |

## 后端接口

| 动作 | 方法 | 路径 | 权限 |
| --- | --- | --- | --- |
| 列表 | GET | `/system/merchant/profile/list` | `system:merchant:profile:list` |
| 详情 | GET | `/system/merchant/profile/{merchantId}` | `system:merchant:profile:query` |
| 修改 | PUT | `/system/merchant/profile` | `system:merchant:profile:edit` |

## 数据表

| 表名 | 用途 |
| --- | --- |
| `merchant_profile` | 商家主体资料。 |
| `sys_tenant` | 商家租户。 |

## 关键规则

- 只有平台租户可以跨商家查询和编辑。
- 当前编辑只更新联系资料、地址和备注。
- 商家名称、公司名称、主邮箱、国家、审核状态不是当前普通编辑字段。

## 可检索性与引用关系

业务编码或单号：

- 当前代码使用 `merchantId` / `tenantId` 作为详情入口，未提供独立商家编码。

搜索展示字段：

- `merchantName`
- `companyName`
- `primaryEmail`
- `country`
- `status`
- `auditStatus`
- `updateTime`

主要引用关系：

- 商家资料通过 `tenantId` 关联商家租户。
- 商户人员管理通过 `/system/merchant/profile/list` 选择目标商家。

权限过滤依据：

- `tenantId`
- `system:merchant:profile:list` / `system:merchant:profile:query`
- `status`
- `auditStatus`

详情入口：

| 入口 | 内容 |
| --- | --- |
| 前端路由 | `/merchantManagement/merchantProfile` |
| 后端详情接口 | `GET /system/merchant/profile/{merchantId}` |

说明：搜索结果只作为入口和摘要展示；商家状态、审核状态和可编辑字段以业务数据库和详情接口为准。

## AI 工具调用契约

### 意图

```text
意图名称：平台更新商家联系资料
调用接口：PUT /system/merchant/profile
权限要求：system:merchant:profile:edit
风险级别：中风险
```

### Payload 边界

- AI 必须先定位唯一商家。
- AI 不提交审核字段和不可编辑主体字段。

### 失败提示

| 场景 | 用户提示 |
| --- | --- |
| 目标商家不唯一 | 请明确要维护哪个商家。 |
| 非平台租户 | 当前账号不能跨商家维护资料。 |
