# 商家人员管理操作手册

## 功能定位

商家人员管理用于平台/厂家租户跨商家维护员工账号。它复用商家用户页面，并开启平台模式。

## 页面入口

- 菜单：商家管理 / 商家人员管理
- 路由：`/merchantManagement/merchantStaff`

## 常用操作

1. 先选择目标商家。
2. 查询该商家的员工。
3. 新增、编辑、删除或启停员工。

平台模式下，如果没有选择商家，新增按钮不可用。

## AI 代办能力

### 支持的意图

```text
意图名称：平台新增商家员工
用户说法：给 ABC Dealer 新增一个员工账号。
```

### 必填槽位

- 目标商家。
- 登录账号。
- 昵称。
- 初始密码。

### 需要追问

- 目标商家无法唯一定位。
- 缺少登录账号、昵称或初始密码。
- 删除、停用、重置密码必须二次确认。

### 保存前确认摘要

```text
将新增商家员工：
- 目标商家：ABC Dealer
- 登录账号：staff@example.com
- 昵称：Sarah
是否确认保存？
```
