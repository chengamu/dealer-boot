# 商户等级操作手册

## 页面入口

商家管理 / 商户等级。

## 常用操作

1. 新增等级：填写等级编码、名称、默认折扣、默认信用额度、排序和状态。
2. 设为默认：勾选默认等级后，后续审核通过的新商家会自动带入该等级。
3. 停用等级：等级不再作为新商家可选项，已有商家资料保留历史值。
4. 删除等级：仅停用且未被引用的等级允许删除。

## 注意事项

- 默认折扣必须在 `0` 到 `1` 之间，默认信用额度不能小于 `0`。
- 默认信用额度只是商户信用配置，不代表钱包余额，不产生收款或占用流水。
- 等级默认折扣是兜底折扣；如果存在等级产品折扣，按等级产品折扣优先。
- 启用中的等级只允许停用，不展示修改和删除操作。
