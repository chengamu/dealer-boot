# 商家审核操作手册

## 功能定位

商家审核用于平台/厂家租户处理公开提交的商家申请。审核通过后系统创建商家租户、商家资料和系统默认部门，并把注册申请人关联到平台固定店主角色。

## 页面入口

- 菜单：商家管理 / 商家审核
- 路由：`/merchantManagement/tenantApplication`
- 兼容路由：`/system/tenant/applications`

## 状态说明

| 状态 | 含义 | 可执行动作 |
| --- | --- | --- |
| `PENDING` | 待审核 | 批准、拒绝 |
| `APPROVED` | 已通过 | 只读 |
| `REJECTED` | 已拒绝 | 只读 |

## 常用操作

1. 查询待审核申请。
2. 打开详情核对商家名称、邮箱、联系人和地址。
3. 批准申请：系统创建租户、资料、默认部门和店主账号，并关联平台固定店主角色。
4. 拒绝申请：填写拒绝原因。

## AI 代办能力

### 支持的意图

```text
意图名称：查询待审核商家申请
用户说法：帮我找一下待审核的商家申请。
```

```text
意图名称：批准商家申请
用户说法：批准 ABC Dealer 的申请。
```

### 需要追问

- 批准或拒绝前必须确认唯一申请。
- 拒绝申请必须追问拒绝原因。

### 保存前确认摘要

```text
将批准商家申请：
- 商家名称：ABC Dealer
- 登录邮箱：owner@example.com
批准后会创建商家租户、商家资料和管理员账号。
是否确认？
```
