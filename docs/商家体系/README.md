## 模块边界

### 商家中心

商家中心是商家租户登录后维护自身资料和内部员工的入口。

当前包含：

- 商家资料：当前商家查看和维护可编辑联系资料。
- 商家用户：当前商家维护本租户员工账号。

不包含：

- 平台审核商家申请。
- 平台跨商家维护资料。
- 经销商下单门户 demo 页面。

### 商家管理

商家管理是平台/厂家租户管理商家主体、申请和商家员工的入口。

当前包含：

- 商家申请审核。
- 商家资料管理。
- 商家员工管理。
- 商家等级。
- 等级产品折扣。

不包含：

- 商家自己的资料维护入口。
- 产品配置、订单、报价、支付流水。

### 客户管理

客户管理是租户客户池入口。普通租户只维护当前租户客户；平台/厂家租户额外拥有全部客户入口，可跨租户查看商家客户和厂家自有客户。

当前包含：

- 客户资料。
- 全部客户。
- 报价管理：按客户和项目维护多行产品配置，复用配方与价格引擎测算并导出客户报价。

不包含：

- 正式订单、收款、信用占用。
- 客户门户或 C 端账号体系。
