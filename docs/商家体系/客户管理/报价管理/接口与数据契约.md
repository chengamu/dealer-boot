# 报价管理接口与数据契约

## 模块定位

报价管理面向平台或商家租户的业务人员。用户为客户和项目录入多条产品尺寸、数量和配方业务选项，系统复用可售产品冻结配方版本、价格设置和邮费模板生成客户报价。

报价不直接读取或展示物料单价、用量公式和 BOM。商家等级折扣属于后续下单付款规则，不参与客户报价；`discount_amount` 当前固定保存为 `0` 且前端和导出不展示。

## 页面与状态

| 页面 | 路由 | 说明 |
| --- | --- | --- |
| 报价列表（平台） | `/customerManagement/quotes` | 平台租户查询、新增、进入工作台、删除草稿、导出已确认报价 |
| 报价列表（商家） | `/merchant/quotes` | 商家租户在自身数据范围内使用同一报价能力 |
| 报价工作台 | `/customer/quotes/workbench?quoteId={id}` | 维护主信息、多行产品配置、逐行或整单测算 |

状态只有 `DRAFT`、`CONFIRMED`、`VOID`：

- 草稿可编辑、测算和删除。
- 确认时服务端重新测算全部明细；全部通过后冻结，不再允许普通编辑。
- 已确认报价可导出或作废；作废后保留历史，不可恢复。
- 有效期由 `valid_until` 判断，不另存过期状态。

## 接口

| 方法 | 路径 | 说明 |
| --- | --- | --- |
| `GET` | `/customer/quotes/list` | 当前租户报价分页列表 |
| `GET` | `/customer/quotes/{id}` | 报价主档和明细 |
| `GET` | `/customer/quotes/catalog/products` | 报价可选的已启用、价格就绪平台产品 |
| `GET` | `/customer/quotes/catalog/products/{saleProductId}/setup` | 报价所需的配方业务选项和值，不返回内部价格、物料规则或 BOM |
| `POST` | `/customer/quotes` | 新增草稿，单号由后端生成 |
| `PUT` | `/customer/quotes` | 保存草稿，服务端重新计算可计算的明细 |
| `DELETE` | `/customer/quotes/{ids}` | 删除草稿 |
| `POST` | `/customer/quotes/calculate-item` | 逐行测算，不落库 |
| `PUT` | `/customer/quotes/{id}/calculate` | 整单测算并保存结果 |
| `PUT` | `/customer/quotes/{id}/confirm` | 严格重算并确认冻结 |
| `PUT` | `/customer/quotes/{id}/void` | 作废已确认报价 |
| `POST` | `/customer/quotes/{id}/export` | 按报价语言导出已确认报价 |

## 价格与邮费

- 报价主档、客户和业务负责人归当前登录租户；可售产品、配方版本、价格设置和邮费模板固定从平台租户 `1` 的目录读取。
- 报价前端只调用 `customer:quote:*` 权限保护的报价目录接口，不授予商家产品管理或价格管理权限。
- 产品目录详情只向前端返回可配置的配方选项和值；价格设置、物料价格规则、用量和 BOM 只在服务端测算链路内部使用。
- 产品单价来自可售产品对应的 `READY` 价格设置；产品金额为单价乘数量。
- 币种来自价格设置，第一版统一为 `USD`，页面不提供币种、汇率和税设置。
- 邮费按价格币种过滤启用模板，再按宽高换算平方英尺，并依据是否选中物料类型 `MOTOR` 的系统配置命中 `MANUAL/MOTORIZED` 规则。
- 邮费面积区间按左闭右开匹配：`minArea <= area < maxArea`；最后一段最大面积为空表示不封顶。
- 单件邮费来自命中规则；邮费总额为单件邮费乘数量。
- 行金额为产品金额加邮费，报价合计为所有行金额之和。
- 同一报价不允许混用币种。

## 双语契约

- `quote_language` 为 `ZH_CN` 或 `EN_US`，只控制客户可见的选项名称、选项值、配置摘要和导出表头。
- 操作界面语言继续跟随当前登录用户语言，不由报价语言改变。
- 选项选择稳定保存 `option_code -> value_code`，切换报价语言不会改变选择结果。
- 英文名称缺失时草稿用稳定编码显式提示；英文报价在确认和导出前必须保证所有客户可见选项和值均有英文名称，不回退显示中文。

## 快照边界

`customer_quote_item` 冗余可售产品、配方版本、邮费模板、双语配置摘要和最终金额。`pricing_snapshot_json` 保存内部价格试算结果以及命中的邮费模板和规则，仅供审计，不进入客户导出文件。已确认报价导出只读取这些冻结字段，不因现行配方、选项或价格设置后续变化而重新解释历史报价。后续转订单应引用确认后的报价明细快照，不重新读取草稿配置。

确认和作废使用报价 ID 业务锁，并通过带原状态条件的更新兜底；并发请求只能有一个成功。`confirmed_time` 使用 `TimeUtils.utcNow()` 写入 UTC 语义时间。

客户、可售产品、配方版本和邮费模板被任何未删除报价或报价明细引用时不可删除；草稿删除后解除引用，已确认或已作废报价保留历史引用。相关主数据可以停用，停用不改写历史快照。

## 权限

权限为 `customer:quote:list/query/add/edit/remove/export`。报价主档的查询、修改、测算、确认和导出均按当前租户隔离；商家读取平台产品目录由报价服务内部显式切换租户，不开放产品管理权限。工作台按权限隐藏保存、测算、确认、作废和导出动作；只有查询权限时页面为只读，不展示点击后必然返回 `403` 的按钮。
