# 订单测算与客户报价接口契约

订单测算与客户报价统一位于“销售管理 > 订单测算”。列表进入测算工作台；确认报价后在同一业务入口输出 Excel、PDF、打印或邮件，并可转为正式销售订单。

## 模块定位

`customer_quote` 与 `customer_quote_item` 是工程项目订单测算和客户报价的唯一底座，不再建立第三套测算模型。一个测算项目可包含多个房间、多个产品和多条尺寸配置。

主链固定为：

```text
订单测算草稿 -> 全部明细测算通过 -> 确认客户报价 -> 转正式销售订单
```

客户报价使用价格设置币种，第一版为 `USD`，税费为零。商家等级折扣不进入客户报价、Excel、PDF 或邮件，只在转销售订单时计算平台订单应付金额。

## 状态与转换

- `DRAFT`：可编辑、复制、删除、逐行测算和整单测算。
- `CONFIRMED`：确认时严格重算并冻结，不可修改；可复制、导出、打印、邮件和转订单。
- `VOID`：保留历史，不可恢复，可复制为新草稿。
- 是否已转订单由 `sales_document_id` 判断，不增加第四种状态。
- 已转订单的报价不可作废、删除或再次转换；订单取消后也只能复制报价形成新草稿。

修改产品、尺寸、数量或配置项后，该行恢复为 `PENDING`，必须重新测算。只有全部明细为 `PASS` 才能确认。

## 快照

确认报价冻结以下内容：

- 可售产品、产品分类和产品类型。
- 配方与正式配方版本。
- 宽度、高度、数量。
- 稳定的选项和值编码，以及中英文配置摘要。
- 产品单价、产品金额、单件邮费和邮费合计。
- 价格设置、命中规则、邮费模板与命中规则。
- `bom_snapshot_json` 中的生产 BOM。

转订单直接复制确认报价快照，不重新调用配方、价格、邮费或 BOM 引擎。唯一实时读取的是商家当前等级折扣。

## 接口

| 方法 | 路径 | 说明 |
| --- | --- | --- |
| `GET` | `/customer/quotes/list` | 当前租户测算分页 |
| `GET` | `/customer/quotes/{id}` | 主档与明细 |
| `POST/PUT/DELETE` | `/customer/quotes`、`/customer/quotes/{ids}` | 草稿新增、保存、删除 |
| `POST` | `/customer/quotes/calculate-item` | 单行测算，不落库 |
| `PUT` | `/customer/quotes/{id}/calculate` | 整单测算并保存 |
| `PUT` | `/customer/quotes/{id}/confirm` | 严格重算并冻结报价 |
| `PUT` | `/customer/quotes/{id}/void` | 作废未转单的确认报价 |
| `POST` | `/customer/quotes/{id}/copy` | 复制为新草稿，不复制测算结果和订单关系 |
| `POST` | `/customer/quotes/{id}/order-preview` | 按当前商家等级预览订单应付金额 |
| `POST` | `/customer/quotes/{id}/convert-order` | 幂等转换为 `SUBMITTED` 销售订单 |
| `POST` | `/customer/quotes/{id}/export` | 按报价语言导出 Excel |
| `GET` | `/customer/quotes/{id}/pdf` | 客户报价 PDF |
| `POST` | `/customer/quotes/{id}/email` | 邮件发送客户报价 PDF |

目录接口 `/customer/quotes/catalog/products` 与 `/catalog/products/{id}/setup` 只返回可售产品和客户可见配置项，不向商家开放产品或价格管理权限。

## 转订单事务

转换以 `quoteId` 使用 `customer-quote-convert` 业务锁并运行在同一事务中：

1. 校验报价已确认且快照完整。
2. 已存在 `sales_document_id` 时直接返回原订单，保证重复请求幂等。
3. 读取商家当前等级折扣；只折扣产品金额，邮费不折扣。
4. 校验前端预览总额 `expectedTotalAmount`；金额变化时不创建订单。
5. 创建 `SUBMITTED` 销售订单和明细，写 `ORDER_CREATED_FROM_QUOTE` 流水。
6. 以 `status=CONFIRMED AND sales_document_id IS NULL` 条件回写报价；失败则整笔回滚。

## 双语与权限

- `quote_language` 为 `ZH_CN` 或 `EN_US`，只控制客户文件中的配置名称和值。
- 操作界面语言跟随登录用户，不受报价语言影响。
- 英文报价确认前，所有客户可见配置项和值必须有英文名称，不回退中文。
- 权限为 `customer:quote:list/query/add/edit/remove/export/document/email/convert`。
- 前端无权限或状态不允许的动作直接隐藏，后端继续做权限与状态拦截。
