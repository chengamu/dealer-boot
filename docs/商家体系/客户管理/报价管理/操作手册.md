# 报价管理操作手册

## 使用前提

开始报价前应确保当前租户客户已启用，并已在平台产品目录完成配方审核、可售产品启用、价格设置就绪和对应币种邮费模板启用。商家报价工作台通过报价专用接口读取平台目录，不需要产品管理或价格管理权限；页面只能取得客户可配置的配方选项和值，内部价格规则、用量和 BOM 不下发。

## 操作顺序

1. 在报价列表新增报价，选择客户、项目、有效期、业务负责人和报价语言。
2. 添加产品行，填写房间或位置、可售产品、宽度、高度和数量。
3. 展开产品行。选项按配方模拟中的父子配置关系分组纵向展示；修改父选项后，只展示当前条件有效的下级选项。
4. 可逐行测算，也可保存草稿后执行整单测算。任一输入或选项变化都会把当前行恢复为待测算。
5. 全部明细显示已测算后确认报价。确认接口仍会重新严格测算，不能依赖浏览器中的旧金额。
6. 已确认报价按 `中文` 或 `English` 导出 Excel；导出内容只包含客户、产品配置和报价金额，并使用确认时冻结的配置摘要与金额。

## 业务说明

- 宽度和高度单位固定为英寸，页面显示两位小数。
- 产品金额和邮费均按数量计算。
- 页面固定显示价格设置币种，当前为美元；不提供税、汇率和商家等级折扣。
- 英文报价必须先补齐配方业务选项和值的英文名称，否则不能确认或导出。
- 确认后若需要调整，应新建新的草稿报价；当前版本不修改已确认快照。
- 同一报价的确认或作废不能并发重复执行；重复操作会按当前真实状态拒绝。
- 已被未删除报价引用的客户、可售产品、配方版本和邮费模板只能停用，不能删除；已作废报价仍保留历史引用。
- 页面只展示当前账号拥有权限的动作；只有查询权限时可以查看报价，但不能保存、测算、确认、作废或导出。
