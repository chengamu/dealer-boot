# 订单测算操作手册

## 使用前提

开始测算前，平台需已完成：配方审核、可售产品启用、价格设置就绪和 `USD` 邮费模板启用。客户资料必须处于启用状态。

## 测算与报价

1. 新建订单测算，选择客户、项目、负责人、有效期和报价语言。
2. 添加产品行，填写房间/位置、产品、宽度、高度和数量。
3. 工程项目可从 Excel 复制四列“房间、宽度、高度、数量”批量加入 Grid，再在系统内选择产品和动态配置项。
4. 可勾选多行复制或删除；复制行会复制当前配置项，但测算结果恢复为待测算。
5. 展开产品行设置业务选项。父选项改变后，失效的子选项由系统清理。
6. 逐行测算或保存后整单测算。任一业务输入变化都会使当前行恢复为待测算。
7. 全部明细通过后确认报价；服务端会再次严格测算并冻结产品、配置、价格、邮费和 BOM 快照。
8. 确认报价可以按报价语言导出 Excel、生成或打印 PDF、发送邮件。

## 转销售订单

1. 在已确认且未转单的报价上点击“转销售订单”。
2. 核对客户报价金额、商家等级、产品原价、等级折扣、邮费和订单应付总额。
3. 填写收货人、电话、地址，可选填写客户 PO 和订单备注。
4. 确认后直接生成 `SUBMITTED` 正式订单并进入订单详情。

客户看到的报价金额不含商家等级折扣。等级折扣只用于平台和商家之间的销售订单产品金额，邮费不打折。

## 修改与历史

- 草稿可编辑和删除。
- 确认报价不可修改；需要调整时复制为新草稿。
- 未转单的确认报价可作废，作废后不可恢复。
- 已转订单报价不可作废或再次转单；即使订单取消，也需复制原报价重新确认。
- 客户、可售产品、配方版本和邮费模板被报价或订单引用后不能删除，可以停用。
