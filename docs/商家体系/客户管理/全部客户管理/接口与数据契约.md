# 全部客户管理接口与数据契约

## 功能定位

全部客户是平台/厂家跨租户查看客户资料的入口，用于监管商家客户和厂家自有客户。

## 接口

| 动作 | 方法 | 路径 | 权限 |
| --- | --- | --- | --- |
| 分页 | GET | `/platform/customers/list` | `platform:customer:list` |
| 负责人选项 | GET | `/platform/customers/owner-options?tenantId={tenantId}` | `platform:customer:list` |
| 详情 | GET | `/platform/customers/{id}` | `platform:customer:query` |

## 查询字段

| 字段 | 说明 |
| --- | --- |
| `merchantId` / `merchantName` | 按商家过滤。 |
| `tenantId` | 内部过滤参数，由前端根据所选商家的租户关系带出，不作为用户可见查询项。 |
| `ownerUserId` | 按负责人过滤。 |
| `email` | 按邮箱模糊查询。 |
| `status` | 按启停状态过滤。 |

## 规则

- 只有平台/厂家租户可访问。
- 全部客户第一版只读，不提供跨租户直接新增或编辑。
- 前端以“归属主体”展示客户来源：商家客户展示 `merchantName`，平台/厂家自有客户展示业务文案，不展示内部租户 ID。
- 负责人选项必须传入目标租户，只返回目标租户启用且未删除用户；目标租户由所选商家带出，界面不要求用户输入租户 ID。
- 非平台租户访问会被后端拒绝。
