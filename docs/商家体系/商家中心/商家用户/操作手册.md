# 商家用户操作手册

## 功能定位

商家用户用于商家租户维护本租户员工账号。它只管理当前商家的员工，不跨商家维护。

## 页面入口

- 菜单：商家中心 / 商家用户
- 路由：`/merchant/users`

## 常用操作

1. 查询员工：按登录账号、手机号、状态、创建时间查询。
2. 新增员工：填写昵称、登录账号、初始密码、手机号、邮箱、性别、状态和备注。
3. 编辑员工：维护基础资料。
4. 启停员工：控制员工是否可继续登录。
5. 删除员工：不能删除当前登录用户，不能删除店主账号。

新增员工会自动写入当前商家租户，由后端分配系统默认部门和“营业员”角色；页面不展示部门或角色选择。

## 使用前准备

- 商家必须已通过审核。
- 当前账号需具备商家用户维护权限。

## AI 代办能力

### 支持的意图

```text
意图名称：新增商家员工
用户说法：给门店新增一个员工账号，邮箱是 staff@example.com，昵称 Sarah。
```

```text
意图名称：停用商家员工
用户说法：把 Sarah 的员工账号停用。
```

### 必填槽位

新增商家员工：

- 登录账号。
- 昵称。
- 初始密码。

### 需要追问

- 缺少登录账号、昵称或初始密码。
- 用户要求删除、停用或重置密码时，必须保存前确认。

### 保存前确认摘要

```text
将新增商家员工：
- 登录账号：staff@example.com
- 昵称：Sarah
- 状态：正常
是否确认保存？
```
