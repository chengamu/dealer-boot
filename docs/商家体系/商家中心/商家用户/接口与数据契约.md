# 商家用户接口与数据契约

## 页面与权限

| 项目 | 内容 |
| --- | --- |
| 菜单 | 商家中心 / 商家用户 |
| 前端路由 | `/merchant/users` |
| 前端组件 | `admin-ui/src/pages/merchant/MerchantUserPage.vue` |
| 权限码 | `merchant:user:*` |

## 后端接口

| 动作 | 方法 | 路径 | 权限 |
| --- | --- | --- | --- |
| 列表 | GET | `/merchant/user/list` | `merchant:user:list` |
| 详情 / 初始化表单 | GET | `/merchant/user/`、`/merchant/user/{userId}` | `merchant:user:query` |
| 新增 | POST | `/merchant/user` | `merchant:user:add` |
| 修改 | PUT | `/merchant/user` | `merchant:user:edit` |
| 删除 | DELETE | `/merchant/user/{userIds}` | `merchant:user:remove` |
| 重置密码 | PUT | `/merchant/user/resetPwd` | `merchant:user:resetPwd` |
| 启停 | PUT | `/merchant/user/changeStatus` | `merchant:user:edit` |

## 数据表

| 表名 | 用途 |
| --- | --- |
| `sys_user` | 商家员工账号。 |
| `sys_role` / `sys_role_menu` | 平台全局维护的店主、营业员固定角色及菜单权限。 |
| `sys_user_role` | 商家用户与平台固定角色的关联。 |
| `sys_dept` | 系统默认部门，不向商家展示或开放选择。 |

## 关键规则

- 商家中心按当前登录租户维护员工。
- 平台固定商户角色只有店主 `merchant_admin` 与营业员 `merchant_employee`；数据范围分别为 `4`（本部门及以下）与 `5`（仅本人），角色和菜单统一使用 `tenant_id=1`。
- 新增员工由后端强制写入当前租户、系统默认部门和营业员角色，忽略客户端提交的租户、部门和角色值。
- 详情 / 初始化表单不返回部门、角色 options 或角色关联字段。
- 不能删除当前登录用户。
- 不能删除店主角色账号。
- 登录账号、手机号、邮箱需要唯一。

## 可检索性与引用关系

业务编码或单号：

- 当前代码使用 `userId` 和 `userName` 作为账号入口，未提供独立商家用户编号。

搜索展示字段：

- `userName`
- `nickName`
- `email`
- `phonenumber`
- `status`

主要引用关系：

- 商家用户通过当前登录 `tenantId` 归属商家租户。
- 账号通过 `sys_user_role` 关联店主或营业员角色。

权限过滤依据：

- 当前登录 `tenantId`
- `merchant:user:list` / `merchant:user:query`
- `status`

详情入口：

| 入口 | 内容 |
| --- | --- |
| 前端路由 | `/merchant/users` |
| 后端详情接口 | `GET /merchant/user/{userId}` |

说明：搜索结果只作为入口和摘要展示；账号状态、角色关系和可操作范围以业务数据库和详情接口为准。

## AI 工具调用契约

### 意图

```text
意图名称：新增商家员工
调用接口：POST /merchant/user
权限要求：merchant:user:add
风险级别：中风险
```

```text
意图名称：停用商家员工
调用接口：PUT /merchant/user/changeStatus
权限要求：merchant:user:edit
风险级别：高风险
```

### Payload 边界

- AI 不提交内部租户 ID，当前租户由登录上下文决定。
- AI 不暴露密码、密码哈希、内部用户 ID。
- 删除、停用、重置密码必须保存前确认。

### 失败提示

| 场景 | 用户提示 |
| --- | --- |
| 登录账号重复 | 登录账号已存在，请换一个账号。 |
| 删除当前用户 | 不能删除当前登录用户。 |
| 删除店主 | 店主账号不能删除。 |
