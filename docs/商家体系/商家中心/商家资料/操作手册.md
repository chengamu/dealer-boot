# 商家资料操作手册

## 功能定位

商家资料是商家租户维护自身联系资料和地址资料的入口。主体字段来自商家申请审核结果，当前不在商家中心开放修改。

页面布局分为四块：顶部带图标的商家摘要、左侧分区导航、中间资料编辑区、右侧资料预览。商家名称、公司名称、主邮箱、国家、等级、折扣和信用额度以只读摘要展示，不再用禁用输入框混在可编辑表单里；联系人和地址字段使用顶部标签表单，避免长英文标签挤压输入框。

## 页面入口

- 菜单：商家中心 / 商家资料
- 路由：`/merchant/profile`

## 字段说明

| 字段类型 | 字段 | 当前行为 |
| --- | --- | --- |
| 锁定字段 | 商家名称、公司名称、主邮箱、国家或地区 | 只读，不允许商家中心修改。 |
| 只读业务字段 | 商家等级、当前折扣、当前信用额度 | 只读，由平台在商家管理中维护。 |
| 可编辑字段 | 联系人名、联系人姓、办公电话、手机、州/省、城市、地址、邮编、备注 | 可由商家保存。 |

保存后系统根据联系人名和联系人姓生成联系人姓名。

当前折扣按两位小数展示，例如 `0.95`。

## 常用操作

1. 进入商家中心 / 商家资料。
2. 查看锁定的主体资料。
3. 修改联系人、电话、地址或备注。
4. 保存后刷新页面确认。

## 常见问题

| 问题 | 原因 | 处理方式 |
| --- | --- | --- |
| 商家名称不能改 | 商家名称来自审核通过的申请资料 | 由平台在商家管理中确认是否调整。 |
| 主邮箱不能改 | 主邮箱是商家登录和审核链路的核心标识 | 后续如需变更，应设计主体资料变更流程。 |
| 等级和折扣不能改 | 等级、折扣和信用额度属于平台定价配置 | 联系平台/厂家在商家管理中维护。 |

## AI 代办能力

### 支持的意图

```text
意图名称：更新当前商家联系资料
用户说法：把我的办公电话改成 020-12345678，联系人改成 Sarah Johnson。
```

### 必填槽位

- 至少一个可编辑字段。

### 需要追问

- 用户要求修改商家名称、主邮箱、国家或审核状态。
- 联系人只有一个完整姓名时，需要确认如何拆分名和姓，或按当前页面规则保存。

### 保存前确认摘要

```text
将更新当前商家资料：
- 联系人：Sarah Johnson
- 办公电话：020-12345678
是否确认保存？
```
