# 商家资料接口与数据契约

## 页面与权限

| 项目 | 内容 |
| --- | --- |
| 菜单 | 商家中心 / 商家资料 |
| 前端路由 | `/merchant/profile` |
| 前端组件 | `admin-ui/src/pages/merchant/MerchantProfilePage.vue` |
| 权限码 | `merchant:profile:query`、`merchant:profile:edit` |

## 后端接口

| 动作 | 方法 | 路径 | 权限 |
| --- | --- | --- | --- |
| 查询当前商家资料 | GET | `/system/merchant/profile/current` | `merchant:profile:query` |
| 修改当前商家资料 | PUT | `/system/merchant/profile/current` | `merchant:profile:edit` |

## 数据表

| 表名 | 用途 |
| --- | --- |
| `merchant_profile` | 商家主体资料，一对一关联商家租户。 |
| `sys_tenant` | 商家租户。 |
| `merchant_level` | 平台维护的商户等级来源，商家中心只读展示。 |

## 可编辑字段

| 字段 | 说明 |
| --- | --- |
| `contact_first_name` / `contact_last_name` | 联系人名和姓。 |
| `contact_name` | 由名和姓拼接。 |
| `office_phone` / `mobile_phone` | 联系电话。 |
| `state` / `city` / `address_line1` / `address_line2` / `postal_code` | 地址信息。 |
| `remark` | 备注。 |

## 只读展示字段

| 字段 | 说明 |
| --- | --- |
| `level_id` / `level_code` / `level_name` | 当前商户等级。 |
| `discount_rate` | 当前折扣。 |
| `credit_limit` | 当前信用额度，本轮只读展示，不接支付或订单占用。 |

## 禁止普通修改字段

- `merchant_id`
- `tenant_id`
- `merchant_name`
- `company_name`
- `primary_email`
- `country`
- `status`
- `audit_status`
- `level_id`
- `level_code`
- `level_name`
- `discount_rate`
- `credit_limit`
- `audit_by`
- `audit_time`

## 可检索性与引用关系

业务编码或单号：

- 当前代码使用 `merchantId` / `tenantId` 作为详情入口，未提供独立商家编码。

搜索展示字段：

- `merchantName`
- `companyName`
- `primaryEmail`
- `country`
- `status`
- `auditStatus`
- `updateTime`

主要引用关系：

- 当前商家资料通过登录上下文中的 `tenantId` 关联商家租户。
- 商家中心只维护当前租户资料，不跨商家检索。

权限过滤依据：

- 当前登录 `tenantId`
- `merchant:profile:query` / `merchant:profile:edit`
- `status`
- `auditStatus`

详情入口：

| 入口 | 内容 |
| --- | --- |
| 前端路由 | `/merchant/profile` |
| 后端详情接口 | `GET /system/merchant/profile/current` |

说明：搜索结果只作为入口和摘要展示；商家主体字段、审核状态和可编辑范围以业务数据库和详情接口为准。

## AI 工具调用契约

### 意图

```text
意图名称：更新当前商家联系资料
调用接口：PUT /system/merchant/profile/current
权限要求：merchant:profile:edit
风险级别：中风险
```

### Payload 边界

- AI 只提交可编辑字段。
- AI 不提交内部 ID、审核字段、主体字段。
- AI 不修改商家名称、主邮箱、国家、等级、折扣、信用额度或审核状态。

### 失败提示

| 场景 | 用户提示 |
| --- | --- |
| 当前账号不是商家租户 | 当前账号不能维护商家中心资料。 |
| 修改锁定字段 | 商家名称、主邮箱和国家地区不能在商家中心修改。 |
