# 商家中心接口与数据契约

## 页面与权限

| 功能 | 前端路由 | 前端组件 | 权限码 |
| --- | --- | --- | --- |
| 商家资料 | `/merchant/profile` | `admin-ui/src/pages/merchant/MerchantProfilePage.vue` | `merchant:profile:query`、`merchant:profile:edit` |
| 商家用户 | `/merchant/users` | `admin-ui/src/pages/merchant/MerchantUserPage.vue` | `merchant:user:*` |

## 后端接口

### 商家资料

| 动作 | 方法 | 路径 | 权限 |
| --- | --- | --- | --- |
| 查询当前商家资料 | GET | `/system/merchant/profile/current` | `merchant:profile:query` |
| 修改当前商家资料 | PUT | `/system/merchant/profile/current` | `merchant:profile:edit` |

### 商家用户

| 动作 | 方法 | 路径 | 权限 |
| --- | --- | --- | --- |
| 列表 | GET | `/merchant/user/list` | `merchant:user:list` |
| 详情 / 初始化表单 | GET | `/merchant/user/`、`/merchant/user/{userId}` | `merchant:user:query` |
| 新增 | POST | `/merchant/user` | `merchant:user:add` |
| 修改 | PUT | `/merchant/user` | `merchant:user:edit` |
| 删除 | DELETE | `/merchant/user/{userIds}` | `merchant:user:remove` |
| 重置密码 | PUT | `/merchant/user/resetPwd` | `merchant:user:resetPwd` |
| 启停 | PUT | `/merchant/user/changeStatus` | `merchant:user:edit` |

## 数据表

| 表名 | 用途 |
| --- | --- |
| `merchant_profile` | 商家主体资料，一对一关联商家租户。 |
| `sys_tenant` | 租户表，商家租户 `tenant_type = MERCHANT`。 |
| `sys_user` | 商家员工账号，按租户隔离。 |
| `sys_role` / `sys_user_role` | 商家默认角色和员工角色关系。 |
| `sys_dept` | 商家默认部门，当前创建员工时使用 `Store` 部门。 |

## 关键字段

### 商家资料

| 字段 | 说明 |
| --- | --- |
| `merchant_id` | 商家资料内部 ID。 |
| `tenant_id` | 商家租户 ID。 |
| `merchant_name` | 商家名称，商家中心不可编辑。 |
| `company_name` | 公司名称，商家中心不可编辑。 |
| `primary_email` | 主邮箱，商家中心不可编辑。 |
| `country` | 国家或地区，商家中心不可编辑。 |
| `contact_first_name` / `contact_last_name` | 可编辑联系人姓名来源。 |
| `contact_name` | 后端或前端由名和姓拼接。 |
| `office_phone` / `mobile_phone` | 可编辑电话。 |
| `state` / `city` / `address_line1` / `address_line2` / `postal_code` | 可编辑地址。 |
| `status` | 商家状态，来自 `sys_normal_disable` 口径。 |
| `audit_status` | 商家审核状态，来自申请审核结果。 |

### 商家用户

| 字段 | 说明 |
| --- | --- |
| `tenant_id` | 商家中心由当前登录租户决定。 |
| `user_name` | 登录账号，商家内唯一。 |
| `nick_name` | 员工昵称，必填。 |
| `password` | 新增和重置密码时使用，不应在文档或日志中暴露。 |
| `email` / `phonenumber` | 联系方式，需唯一性校验。 |
| `status` | `1` 正常，`0` 停用。 |

## 校验规则

- 只有商家租户账号可以调用当前商家资料查询和修改接口。
- 商家中心修改资料只能修改联系资料、地址和备注，不能修改商家名称、主邮箱、国家或审核状态。
- 商家用户新增时自动写入当前租户、默认 `Store` 部门和 `merchant_employee` 角色。
- 商家用户不能删除当前登录用户。
- 商家用户不能删除商家管理员角色账号。
- 用户名、手机号、邮箱需要唯一。

## AI 工具调用契约

### 意图

```text
意图名称：更新当前商家联系资料
调用接口：PUT /system/merchant/profile/current
权限要求：merchant:profile:edit
```

```text
意图名称：新增商家员工
调用接口：POST /merchant/user
权限要求：merchant:user:add
```

### Payload 边界

- AI 不提交 `tenantId`、`merchantId`、`auditStatus`、`auditBy`、`auditTime`、`createBy`、`createTime`、`updateBy`、`updateTime`。
- AI 不返回或复述密码、验证码、内部用户 ID。
- AI 不修改商家名称、主邮箱、国家或审核状态。
- 删除、停用、重置密码必须保存前确认。

### 失败提示

| 场景 | 用户提示 |
| --- | --- |
| 当前账号不是商家租户 | 当前账号不能维护商家中心资料。 |
| 修改锁定字段 | 商家名称、主邮箱和国家地区不能在商家中心修改。 |
| 登录账号重复 | 登录账号已存在，请换一个账号。 |
| 手机号或邮箱重复 | 手机号或邮箱已被使用，请检查后再保存。 |
| 删除管理员 | 商家管理员账号不能删除。 |
