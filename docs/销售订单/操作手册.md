# 销售订单操作手册

## 创建来源

销售订单不能手工新增。业务人员在订单测算中确认客户报价、预览商家等级折扣并补齐收货信息后，直接生成 `SUBMITTED` 正式订单。

## 履约顺序

1. 平台在销售订单列表核对来源报价、客户、商家、收货地址和订单应付金额。
2. 按权限确认付款方式、金额、流水号和付款凭证。
3. 付款后开始生产，完成后生成生产清单 PDF；生产清单只使用订单冻结 BOM，不展示价格。
4. 填写承运商和运单号发货。
5. 商家或有权限人员确认签收，订单进入 `COMPLETED`。

订单 PDF 展示平台销售订单应付金额；客户报价 PDF、Excel 和邮件仍从订单测算模块生成，不展示商家等级折扣。

## 取消与重建

- 未进入受限履约阶段的订单可以取消，取消后不可恢复。
- 取消订单不会释放原报价再次转单。
- 需要重建时，在订单测算中复制原报价为新草稿，重新测算、确认并转换。

页面只展示当前账号有权限且当前状态可执行的动作。重复点击和并发请求由统一业务锁与旧状态条件更新拦截。
