# 销售订单操作手册

## 商家

1. 新建销售单并选择启用客户。
2. 添加可售产品，填写房间、宽、高、数量和业务配置。
3. 测算每一行或整单，全部通过后保存。
4. 生成客户报价，或直接提交订单。
5. 报价可生成 PDF、打印、导出 Excel 或发送邮件；已报价可撤回后修改。
6. 订单发货后确认签收，单据进入完成状态。
7. 在客户资料中可直接查看该客户的销售历史并进入订单详情。

## 平台

1. 在销售单列表查询已提交订单。
2. 核对订单金额并填写付款方式、金额、流水号和付款凭证。
3. 确认付款后开始生产，生产完成后填写承运商和运单号发货。
4. 生产人员可以生成生产清单 PDF，生产清单逐项读取订单冻结的 BOM 快照，不受后续配方和物料变化影响。

订单只支持一次付款、一次生产和一次发货。已付款或已开始生产的订单不能取消。

同一销售单的报价、提交和履约动作由系统串行处理；重复点击或并发请求不会覆盖付款、生产或物流信息。页面只展示当前账号拥有权限且当前状态允许的动作。
