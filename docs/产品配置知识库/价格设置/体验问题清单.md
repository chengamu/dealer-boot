# 价格设置体验问题清单

## 当前已确认

- 价格设置是独立工作台，不使用抽屉。
- 工作台顶部对齐配方设置页的信息密度和操作方式。
- 面料价格不从物料管理手工新增，必须由当前配方版本快照同步生成面料清单。
- 面料价格主表按“一行面料”展示，价格汇总展示默认规则和条件规则。
- 条件价格用于表达尺寸、面积或订单选项命中时的不同单价，不再生成“面料 x 选项组合”的价格矩阵。
- 同步配方面料时需要清理旧的无归属面料价格规则，避免重复生成脏数据。
- 邮费独立于产品价格设置；价格设置不展示、不导入、不校验邮费。
- 下单试算读取当前启用邮费模板，订单提交时冻结命中的邮费金额和来源规则。
- 可售产品启用后，价格规则只读。
- 数字价格按两位小数展示，日期展示到分钟。

## 后续观察

- 是否需要面料价格支持 Excel 批量导入。
- 是否需要把价格校验问题定位到具体表格行。
- 是否需要在后续订单模块中增加价格快照详情页。

## 暂不处理

- 本模块不做订单试算页。
- 本模块不生成具体尺寸 SKU。
- 本模块不维护普通物料销售价，普通物料销售价继续来自物料管理。
