# 价格设置体验问题清单

## 当前已落实

- 价格设置为独立工作台，主表铺满可用高度。
- 全部配方物料自动同步，不提供脱离配方的新增按钮。
- 主表按物料展示，多行汇总默认和条件价格。
- 单个设置和批量设置复用同一条件编辑器、价格公式编辑器和表格结构。
- 批量设置先勾选物料，一次只允许同一分组，并通过单接口原子保存。
- 条件内部使用 `optionRefKey/valueRefKey`，编码改名不会破坏规则。
- 价格设置 API 中参与路由、选择和保存的 19 位主键在前端按字符串传递。
- 同步和覆盖重建均有危险操作确认。
- V1/V2 价格隔离，新版本复制可匹配物料的全部规则。
- 邮费模板独立维护，价格设置不复制邮费快照。

## 下游链路

- 订单测算调用价格试算并在确认客户报价时冻结配方版本、物料价格规则和计算结果。
- 客户报价转销售订单时复制冻结价格快照，不重新调用价格引擎。
- 商家等级折扣只作用于销售订单产品金额，不改写客户报价快照。
