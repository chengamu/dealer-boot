# 价格设置操作手册

## 使用入口

从可售产品列表选择一条记录，点击“价格设置”进入工作台。也可以直接打开：

`/product-pricing/price-settings?saleProductId={可售产品ID}`

## 工作台结构

- 顶部：切换可售产品，查看绑定配方、版本、尺寸范围和价格状态。
- 物料分组概览：展示当前配方版本快照中的面料、铝材、系统、配件、包装数量。
- 面料价格：按当前配方版本快照生成面料价格矩阵，维护不同面料在不同订单选项组合下的单价。
- 邮费公式：维护不带电和带电两条邮费公式。
- 校验问题：展示价格校验结果。

## 面料价格维护

1. 点击“生成当前版本价格”，系统按当前可售产品绑定的配方版本快照生成面料价格矩阵。
2. 矩阵行来自配方快照中的面料原料池，不能在价格设置里手工新增面料。
3. 矩阵列来自配方快照中的订单选项组合，例如“柔式拉珠”“柔式电动”。
4. 在单元格维护对应单价，价格统一保留两位小数展示。
5. 在面料价格公式中维护计算公式，例如 `unitPrice * MAX(drop * 2.54 * width * 2.54 / 10000, 1)`。
6. 需要批量调整时，使用“批量设置价格”按面料范围和选项组合批量写入单价/公式。
7. 保存后价格状态回到未就绪，需要重新校验。

## 配方版本变化

- 可售产品绑定 V1 配方时维护的是 V1 价格。
- 后续绑定 V2 配方时，系统生成新的 V2 价格，不覆盖 V1。
- V2 生成价格时，同面料、同选项组合可复制上一版本价格；新增面料或新增组合需要用户补充价格。

## 邮费维护

1. 系统固定展示“不带电邮费”和“带电邮费”两条规则。
2. 不带电/带电由订单选项是否选择电机类系统判断，不在价格设置里自由录入触发条件。
3. 两条规则都使用公式录入，支持 `width`、`drop`、`MAX`、`MIN`、`ROUND` 等变量和函数。

## 校验与启用

1. 保存面料价格和邮费公式。
2. 点击“校验价格”。
3. 校验无错误后，可售产品价格状态变为“已就绪”。
4. 返回可售产品列表启用该可售产品。

## 冻结规则

可售产品启用后，价格设置只读。需要调整价格时，先停用可售产品，再维护价格并重新校验。
