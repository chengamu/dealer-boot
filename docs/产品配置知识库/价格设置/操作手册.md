# 价格设置操作手册

## 使用入口

从可售产品列表选择一条记录，点击“价格设置”进入工作台。也可以直接打开：

`/product-pricing/price-settings?saleProductId={可售产品ID}`

## 工作台结构

- 顶部：切换可售产品，查看绑定配方、版本、尺寸范围和价格状态。
- 物料分组概览：展示当前配方版本快照中的面料、铝材、系统、配件、包装数量。
- 面料价格：按当前配方版本快照同步面料清单，维护每个面料的默认价格和条件价格。
- 邮费快照：从邮费模板导入当前价格版本的不带电/带电邮费公式。
- 校验问题：展示价格校验结果。

## 面料价格维护

1. 点击“同步配方面料”，系统按当前可售产品绑定的配方版本快照同步面料价格行。
2. 面料价格行来自配方快照中的面料原料池，不能在价格设置里手工新增面料。
3. 主表“价格汇总”展示该面料的默认价格规则和条件价格规则，类似配方原料里的“用量汇总”。
4. 点击“价格设置”维护单个面料的默认规则和条件规则。条件规则可用于“宽度大于 20”“面积小于 50”“系统选择电机”等不同场景。
5. 价格公式用于计算面料销售价，单价在规则字段里维护，公式里展示实际单价，例如 `50 * MAX(drop * 2.54 * width * 2.54 / 10000, 1)`。
6. 需要批量调整时，先在面料价格主表勾选面料，再点击“批量设置价格”进入批量价格规则抽屉；抽屉顶部展示已选面料，下方维护一组默认/条件价格规则，保存后覆盖写入已选面料。
7. 保存后价格状态回到未就绪，需要重新校验。

## 条件和公式展示

- 页面优先展示中文业务名称，例如“宽度(in)”“高度(in)”“面积(㎡)”“系统 = 电机”。
- 表达式内部仍使用 `width`、`drop`、`areaM2`、`option_<optionRefKey>` 等稳定计算 token，保存和校验以表达式为准。
- 价格条件通过结构化条件对象保存，配置项和值必须用配方版本快照中的 `optionRefKey/valueRefKey` 关联；配置项编码、值编码和名称只做展示快照。

## 配方版本变化

- 可售产品绑定 V1 配方时维护的是 V1 价格。
- 后续绑定 V2 配方时，系统生成新的 V2 价格，不覆盖 V1。
- V2 生成价格时，同面料可复制上一版本的默认/条件价格；新增面料需要用户补充价格。

## 邮费维护

1. 先在“邮费模板”维护可复用的邮费公式。
2. 回到价格设置，在邮费区域选择已启用模板并点击“导入邮费模板”。
3. 导入会覆盖当前可售产品 + 当前配方版本下已有邮费快照，不影响其它配方版本价格。
4. 不带电/带电由订单选项是否选择电机类系统判断，不在价格设置里自由录入触发条件。
5. 邮费公式支持 `width`、`drop`、`areaM2`、`areaSqft`、`MAX`、`MIN`、`ROUND` 等变量和函数，不使用面料单价变量。

## 校验与启用

1. 保存面料价格，并导入邮费模板。
2. 点击“校验价格”。
3. 校验无错误后，可售产品价格状态变为“已就绪”。
4. 返回可售产品列表启用该可售产品。

## 冻结规则

可售产品启用后，价格设置只读。需要调整价格时，先停用可售产品，再维护价格并重新校验。
