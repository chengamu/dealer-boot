# 价格设置操作手册

## 正确顺序

1. 配方审核通过并生成生效版本。
2. 可售产品绑定该配方版本。
3. 进入价格设置，点击“同步配方物料”。
4. 维护当前配方快照中的全部物料价格，包括面料、铝材、系统、配件、零件包、包装、印刷品以及后续新增物料分组。
5. 点击“校验价格”，通过后再启用可售产品。

## 物料价格

- 价格主表一行对应当前配方版本中的一个物料，不能手工新增物料。
- “价格汇总”多行展示默认规则和条件规则。
- 面料通常使用面积变量，例如 `50 * MAX(areaM2, 1)`。
- 其他物料通常使用配方用量，例如 `3.5 * usageQty`；用量来自配方原料中的用量规则。
- 条件可使用尺寸、面积和业务选项，例如“宽度 > 20”或“系统 = 电机”。
- 单价是独立字段；编辑器保存时公式可展示实际数字，不要求用户理解 `unitPrice` token。

## 批量设置

1. 在主表勾选需要统一规则的物料。
2. 点击“批量设置价格”。
3. 一次只能选择同一物料分组，避免面积公式和用量公式混用。
4. 在宽抽屉中维护一组默认/条件规则并保存。

批量保存是单次事务：任何一条失败，整批不写入。

## 配方版本变化

- V1 和 V2 各自拥有独立价格设置。
- V2 首次同步时，相同物料自动复制 V1 的全部默认和条件规则。
- V2 新增物料生成待确认默认规则；V1 历史价格不被修改。
- 普通同步会移除已不属于当前版本快照的物料，因此操作前必须确认。

## 邮费

邮费不导入价格设置，也不参与产品价格就绪校验。当前模块只计算物料价格；邮费由邮费模板独立维护，客户报价和销售订单在测算时读取并冻结命中的模板、条件、面积区间和金额。

## 状态

- 可售产品停用：可同步、编辑、批量设置和校验价格。
- 可售产品启用：价格设置只读。
- 任意价格变更后状态回到未就绪，必须重新校验。
