- 有稳定业务编码、单号、名称或摘要，后续可能被全局搜索命中。
- 会被其他模块引用，或会引用其他主数据、单据、文档。
- 搜索结果需要跳转到详情页、编辑页或审核页。
- 搜索结果需要按租户、状态、删除标记或权限过滤。
