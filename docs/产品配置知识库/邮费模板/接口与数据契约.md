# 邮费模板接口与数据契约

## 模块定位

邮费模板维护美国销售场景下可复用的邮费公式。价格设置不会直接编辑邮费公式，只能从已启用模板导入当前可售产品和配方版本的邮费快照。

## 页面路由

| 页面 | 路由 | 说明 |
| --- | --- | --- |
| 邮费模板 | `/product-pricing/shipping-templates` | 标准 grid，维护模板和面积分段公式 |

## 接口清单

| 方法 | 路径 | 说明 |
| --- | --- | --- |
| `GET` | `/product-pricing/shipping-templates/list` | 分页查询邮费模板 |
| `GET` | `/product-pricing/shipping-templates/options` | 查询启用模板选项 |
| `GET` | `/product-pricing/shipping-templates/{id}` | 查询模板详情和规则 |
| `POST` | `/product-pricing/shipping-templates` | 新增模板 |
| `PUT` | `/product-pricing/shipping-templates` | 修改模板 |
| `DELETE` | `/product-pricing/shipping-templates/{ids}` | 删除未被价格设置引用的模板 |
| `PUT` | `/product-pricing/shipping-templates/change-status/{id}/{status}` | 启用或停用模板 |

## 表结构契约

### `pc_shipping_template`

| 字段 | 说明 |
| --- | --- |
| `shipping_template_id` | 主键 |
| `template_code/template_name` | 模板编码和名称 |
| `currency_code` | 币种，默认 `USD` |
| `default_flag` | 默认模板标记 |
| `status` | `ENABLED`、`DISABLED` |
| `sort_order/remark` | 排序和备注 |

### `pc_shipping_template_rule`

| 字段 | 说明 |
| --- | --- |
| `shipping_template_rule_id` | 主键 |
| `shipping_template_id` | 所属模板 |
| `fee_code/fee_name` | `MANUAL` 不带电、`MOTORIZED` 带电 |
| `min_area_sqft/max_area_sqft` | 面积区间，平方英尺；最大面积为空表示不封顶 |
| `formula_text` | 邮费公式 |
| `sort_order/remark` | 排序和备注 |

## 业务规则

- 模板编码唯一，由 Service 控制。
- 租户字段由登录请求上下文注入，新增和修改接口不接受前端指定租户。
- 启用模板必须同时包含 `MANUAL` 和 `MOTORIZED` 两类规则。
- 同一场景下面积区间不能重叠。
- 最大面积为空的不封顶规则必须是同一场景的最后一段。
- 邮费公式不使用面料单价变量，只支持订单宽高、面积变量和 `MAX/MIN/ROUND/CEIL/FLOOR` 等函数。
- 已被价格设置快照引用的模板不能删除，只能停用。
- 价格设置页自动预选已启用的默认模板，但仍由用户确认后导入，不自动覆盖邮费快照。

| 权限 | 说明 |

| `product:shipping-template:add` | 新增邮费模板 |

| `product:shipping-template:remove` | 删除邮费模板 |
| `product:shipping-template:export` | 导出邮费模板 |
