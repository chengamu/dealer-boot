# 邮费模板操作手册

## 使用入口

进入“产品定价 / 邮费模板”维护可复用邮费公式。价格设置页只选择模板并导入快照。

## 维护步骤

1. 新增模板，填写模板编码、模板名称和币种。
2. 至少维护两类公式：不带电、带电。
3. 如邮费随面积阶梯变化，在同一场景下新增多条面积区间规则。
4. 使用“邮费公式编辑器”输入公式，优先使用面积变量 `areaSqft` 或 `areaM2`。
5. 保存并启用模板。
6. 到价格设置页选择已启用模板，导入到当前可售产品的当前配方版本价格；如已配置默认模板，页面会自动预选该模板。

## 公式说明

- `width`：订单宽度，单位 inch。
- `drop`：订单高度，单位 inch。
- `widthCm/dropCm`：厘米宽高。
- `areaM2/areaSqft`：订单面积。
- `MAX(a,b)`：取较大值，常用于最低计费面积。
- `MIN(a,b)`：取较小值，常用于封顶。
- `ROUND/CEIL/FLOOR`：四舍五入、向上取整、向下取整。

示例：`18 + MAX(areaSqft - 20, 0) * 0.35` 表示 20 平方英尺以内收 18 美元，超过部分每平方英尺加 0.35 美元。

## 导入到价格设置

- 导入会覆盖当前可售产品 + 当前配方版本下已有邮费快照。
- 导入不会影响其它可售产品，也不会影响同一可售产品的其它配方版本价格。
- 模板后续修改不会自动反写历史价格；需要用户再次导入，保持版本价格可追溯。
