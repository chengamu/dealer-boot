# 邮费模板操作手册

## 使用入口

进入“产品定价 / 邮费模板”维护可复用邮费价目表。邮费独立于产品价格设置。

## 维护步骤

1. 新增模板。模板编码由系统在保存时自动生成，只需填写模板名称并从标准币种中选择币种。
2. 分别维护“系统 = 电机（带电）”和“系统 ≠ 电机（不带电）”两类适用条件的面积区间和固定邮费。
3. 如邮费随面积阶梯变化，在同一适用条件下新增多条面积区间；最大面积留空表示不封顶。
4. 明细较多时点击“导入表格”，先下载模板，再上传 Excel。导入会替换抽屉内尚未保存的明细。
5. 检查每条区间和邮费后保存模板；新模板默认停用，确认完整后在列表状态开关中启用。
6. 在列表启用新模板后，同币种旧模板自动停用；客户报价测算按价格币种读取当前启用模板。

## 面积阶梯示例

- 不带电，`[0, 20) ft²`：邮费 18 美元。
- 不带电，`[20, 40) ft²`：邮费 25 美元。
- 不带电，`[40, +∞) ft²`：邮费 32 美元。
- 带电使用同样的面积区间维护另一套固定邮费。

方括号表示包含边界，圆括号表示不包含边界。因此面积正好为 `20 ft²` 时命中第二段，不会同时命中两段。

## 适用条件

- “带电/不带电”不是额外录入的产品属性，也不是邮费模板的场景字段。
- 订单配置项选择“系统 = 电机”时使用带电邮费；未选择电机时使用不带电邮费。
- 页面使用固定中文条件展示，底层保存稳定业务码，操作人员不需要手写条件表达式或邮费公式。

## 报价与订单接入

- 客户报价已经按价格币种读取平台当前启用模板。
- 报价配置项选择电机时命中带电邮费，否则命中不带电邮费；再按面积命中价格区间，单件邮费乘数量计入整行金额。
- 报价确认后冻结模板、命中规则和最终邮费；模板后续调价不影响历史报价。
- 被未删除报价明细引用的模板不可删除，可以停用。

- 销售订单按价格币种读取平台当前启用模板。
- 订单配置选择电机时命中带电邮费，否则命中不带电邮费；再按订单面积命中价格区间。
- 订单提交时保存最终邮费及模板、规则来源快照，模板后续变更不影响历史订单。
- 已被报价或销售订单引用的模板不能删除，可以停用并新建替代模板。
