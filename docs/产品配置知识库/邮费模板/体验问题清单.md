# 邮费模板体验问题清单

## 当前已确认

- 邮费模板使用标准 grid 列表和抽屉表单。
- 邮费公式编辑器是价格模块专用实现，不复用配方原料的用量编辑器。
- 页面不展示费用类别、自由触发条件和状态列；带电/不带电由场景字段表达。
- 面积区间使用平方英尺，最大面积为空表示不封顶。
- 价格设置导入模板时必须确认，因为会覆盖当前版本邮费快照。

## 后续观察

- 是否需要支持邮费模板 Excel 导入。
- 是否需要按国家或承运商扩展多个模板维度。
- 是否需要在价格校验问题中定位到具体邮费模板规则。

## 暂不处理

- 本模块不做实际物流渠道报价。
- 本模块不和支付、订单、承运商 API 联动。
