# 可售产品操作手册

## 正确使用顺序

1. 确认基础信息、物料和配方已经维护完成。
2. 在配方管理中完成配方审核，使配方进入生效状态。
3. 进入可售产品，新增可售产品并选择生效配方。
4. 进入价格设置，维护当前配方版本的全部物料价格规则。
5. 校验价格通过后，再启用可售产品。

## 录入说明

- 可售产品名称面向下单选择，应使用业务可识别名称，例如 `Zebra Shade`、`Roman Shade`。
- 产品分类、产品类型、尺寸范围、配方版本由系统从生效配方带出。
- 可售产品不是具体尺寸 SKU；具体 SKU 后续由订单成功后按尺寸、配置和价格快照生成。
- 价格未就绪时不能启用可售产品。

## 状态说明

| 状态 | 说明 |
| --- | --- |
| `DISABLED` | 未启用，可编辑和维护价格 |
| `ENABLED` | 可下单选择，不可普通编辑，不可修改价格规则 |

价格状态：

| 状态 | 说明 |
| --- | --- |
| `NOT_READY` | 未维护或未校验价格 |
| `READY` | 价格校验通过，可启用 |
| `WARNING` | 价格校验存在问题，不能启用 |

## 常见操作

- 新增：填写可售产品编号、名称，选择已生效配方。
- 修改：仅停用状态可普通修改。
- 进入价格设置：选中行后点击“价格设置”。
- 启用：价格校验通过后，在状态列启用。
- 删除：仅未启用且无价格设置、物料价格规则或未删除报价明细引用的数据可删除。
