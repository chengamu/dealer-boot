# 单位管理操作手册

## 功能定位

单位管理用于维护基础信息、配方管理和后续已确认业务模块中使用的业务单位。

它解决的是同一套系统里“尺寸、面积、重量、数量”等单位统一的问题，避免物料、物料属性、工程规则各自手填单位导致混乱。

业务顺序：

```text
单位管理 -> 物料管理 / 物料属性 -> 配方管理 / 已确认的后续业务模块
```

## 使用对象

- 主要使用人：技术科、基础资料维护人员。
- 辅助使用人：实施人员、测试人员。
- 不建议使用人：普通销售、客户。

## 这个功能维护什么

单位管理维护的是系统可选单位，例如：

```text
PCS  个
SET  套
M    米
CM   厘米
MM   毫米
SQM  平方米
KG   千克
GSM  克每平方米
```

后续物料管理里的主单位、副单位，以及面料厚度、门幅、重量等属性，都应优先从这里选择。

## 字段说明

| 字段 | 是否必填 | 怎么填写 | 注意事项 |
| --- | --- | --- | --- |
| 单位编码 | 是 | 稳定编码，例如 `CM`、`M`、`PCS` | 保存后会被物料和工程规则引用，不建议随意修改 |
| 单位名称 | 是 | 中文名称，例如 `厘米` | 给业务人员识别 |
| 单位英文名称 | 否 | 英文名称，例如 `Centimeter` | 用于英文环境 |
| 单位类型 | 是 | 从固定类型中选择 | 用于过滤不同业务场景能选的单位 |
| 精度小数位 | 是 | 当前单位允许的数量小数位 | 范围 0 到 6；计件单位必须为 0 |
| 舍入方式 | 是 | 六种标准舍入方式 | 用于数量换算和配方最终用量舍入 |
| 基准单位 | 否 | 选择同类型下的标准单位 | 用于单位换算 |
| 换算率 | 否 | 当前单位换算到基准单位的比例 | 例如 1 CM = 0.01 M，则换算率为 `0.01` |
| 排序 | 否 | 数字越小越靠前 | 只影响展示顺序 |
| 备注 | 否 | 说明用途 | 建议写业务用途，不写临时沟通内容 |

## 单位类型从哪里来

单位类型用于把单位分成长度、数量、面积、重量等大类。

目标口径：单位类型应来自统一字典或配置项，页面只读取可选值，不应该散落在前端代码里写死。

页面从 `product_unit_type` 产品字典读取单位类型。

第一阶段常用类型：

| 单位类型 | 含义 | 示例 |
| --- | --- | --- |
| 长度 | 尺寸、门幅、厚度、轨道长度等 | `M`、`CM`、`MM` |
| 数量 | 件、套、卷等计数单位 | `PCS`、`SET`、`ROLL` |
| 面积 | 面料面积、计价面积等 | `SQM` |
| 重量 | 重量或克重相关 | `KG`、`GSM` |

后续如果需要体积等类型，应先确认业务场景，再扩展系统枚举。

## 基准单位和换算率是什么意思

基准单位是同一单位类型下的标准换算单位。

换算率表示：

```text
1 当前单位 = 换算率 * 基准单位
```

示例：

| 当前单位 | 基准单位 | 换算率 | 含义 |
| --- | --- | --- | --- |
| M | M | 1 | 1 米 = 1 米 |
| CM | M | 0.01 | 1 厘米 = 0.01 米 |
| MM | M | 0.001 | 1 毫米 = 0.001 米 |
| PCS | PCS | 1 | 1 个 = 1 个 |
| SET | SET | 1 | 1 套 = 1 套 |
| SQM | SQM | 1 | 1 平方米 = 1 平方米 |

普通维护人员一般只需要按已有示例维护，不需要每天理解换算公式。

基准单位不是字典项，而是从单位管理本身选择出来的一个单位。例如 `CM` 的基准单位选择 `M`，`MM` 的基准单位也选择 `M`。

## 推荐录入口径

1. 先维护基础单位，例如 `M`、`PCS`、`SQM`。
2. 再维护派生单位，例如 `CM`、`MM`。
3. 派生单位选择同类型基准单位，并填写换算率。
4. 不确定换算口径时，不要随意新增单位，先找技术科确认。

## 对后续业务的影响

单位管理会影响：

- 物料管理：主单位、副单位和业务展示单位。
- 物料属性：厚度、门幅、重量等属性的单位。
- 物料属性：面料门幅、厚度、克重等属性单位。
- 配方管理：原料池和用量规则中展示和计算单位。
- 后续已确认业务模块：读取稳定单位编码和名称。
- 后续 AI 代办：AI 需要识别“厘米、CM、公分”是否对应同一个单位。

## 常见问题

| 问题 | 原因 | 处理方式 |
| --- | --- | --- |
| 不知道单位类型在哪里设置 | 当前单位类型是系统固定枚举 | 后续页面应加说明或提示 |
| 不知道基准单位是什么 | 这是单位换算字段 | 同类型单位选择一个标准单位，例如 CM 的基准单位是 M |
| 不知道换算率怎么填 | 换算率表示当前单位到基准单位的比例 | 1 CM = 0.01 M，所以 CM 的换算率是 0.01 |
| 不知道舍入方式怎么选 | 舍入方式用于生产用量和单位换算 | 普通生产换算可先用四舍五入；检验数据、整包采购和商业金额按各自专项规则，不能共用一个默认值 |

## AI 代办能力

### 支持的意图

```text
意图名称：新增单位
用户说法：帮我新增一个卷单位 ROLL
```

```text
意图名称：停用单位
用户说法：把这个不用的单位停用
```

### 必填槽位

- 单位编码。
- 单位中文名称。
- 单位类型。

### 可推断字段

- 状态默认启用。
- 普通生产换算的舍入方式可默认四舍五入；检验、采购包装和商业金额必须按专项规则确定。
- 计件单位精度默认 0；计量数量按单位精度录入和展示，不得全局按整数或两位小数处理。完整规则见 [数值精度、展示与舍入规范](../../../项目配置和代码风格/numeric-precision-standards.md)。
- 如果当前单位就是基准单位，基准单位可等于自己，换算率为 1。

### 需要追问

- 用户只说“新增一个单位”，但没有说明单位类型时，需要追问。
- 用户新增派生单位但没有说明基准单位和换算率时，需要追问或给出建议。

### 保存前确认摘要

```text
将新增单位：
- 单位编码：CM
- 单位名称：厘米
- 单位类型：长度
- 基准单位：M 米
- 换算率：0.01
是否确认保存？
```
