## 单位类型

目标口径：单位类型应由统一字典或配置项提供，页面通过选择器读取，不应在页面组件里写死。

建议字典类型：

```text
product_unit_type
```

建议第一阶段支持：

| 值 | 展示 | 说明 |
| --- | --- | --- |
| `LENGTH` | 长度 | 宽度、高度、门幅、厚度、轨道等 |
| `COUNT` | 数量 | 件、套、卷等 |
| `AREA` | 面积 | 平方米等 |
| `WEIGHT` | 重量 | 千克、克重等 |
| `VOLUME` | 体积 | 立方米、升等 |

当前种子数据和 `product_unit_type` 字典使用 `COUNT` 表示数量类单位；前端从基础字典读取单位类型。

## 舍入方式

| 值 | 展示 | 说明 |
| --- | --- | --- |
| `HALF_UP` | 四舍五入 | 默认推荐 |
| `UP` | 向上取整 | 需要保守取大时使用 |
| `DOWN` | 向下取整 | 需要截断时使用 |

## 换算规则

换算规则：

```text
1 当前单位 = conversion_rate * base_unit_code
```

示例：

```text
CM.base_unit_code = M
CM.conversion_rate = 0.01
表示 1 CM = 0.01 M
```

```text
MM.base_unit_code = M
MM.conversion_rate = 0.001
表示 1 MM = 0.001 M
```

## 选择器来源

| 页面字段 | 数据来源 | 过滤条件 |
| --- | --- | --- |
| 单位类型 | 建议 `product_unit_type` 字典 | 启用状态 |
| 舍入方式 | 前端固定枚举 | 第一阶段不是独立维护页 |
| 基准单位 | `/product-capability/units/options` | 默认选择启用单位，建议按同单位类型过滤 |
| 物料单位 | `/product-capability/units/options` | 默认选择启用单位 |
| 面料属性单位 | `/product-capability/units/options` | 应按长度、面积、重量等业务语义过滤 |

## 校验规则

- `unitCode` 必填。
- `unitNameCn` 必填。
- 同一租户下未删除单位的 `unitCode` 不能重复。
- `conversionRate` 应大于 0。
- `baseUnitCode` 应优先选择同一单位类型下的单位。
- 停用或删除前应检查物料、物料属性等引用。

## AI 工具调用契约

### 意图

```text
新增单位
编辑单位
停用单位
查询单位引用
```

### 调用接口和权限

| 意图 | 方法 | 路径 | 权限 |
| --- | --- | --- | --- |
| 新增单位 | POST | `/product-capability/units` | `product:unit:add` |
| 编辑单位 | PUT | `/product-capability/units` | `product:unit:edit` |
| 停用单位 | PUT | `/product-capability/units/change-status/{id}/{status}` | `product:unit:edit` |
| 查询单位引用 | GET | `/product-capability/units/{id}/references` | `product:unit:reference` |

### 必填槽位

- `unitCode`
- `unitNameCn`
- `unitType`

### 可推断字段

- `status = ENABLED`
- `roundingMode = HALF_UP`
- 数量类单位 `precisionScale = 0`
- 自身作为基准单位时 `baseUnitCode = unitCode` 且 `conversionRate = 1`

### 需要追问

- 用户没有说明单位类型时，需要追问。
- 用户新增派生单位但没有说明基准单位和换算率时，需要追问。
- 单位编码已存在时，需要追问是编辑已有单位还是换编码新建。

### Payload 边界

- 不要求用户提供 `unitId` 或审计字段。
- 基准单位用单位编码或名称匹配，保存时由系统使用稳定编码。
- 涉及换算率时必须在保存前确认换算方向。

### 禁止暴露字段

- `unitId`
- `tenantId`
- `delFlag`
- `createBy`
- `createTime`
- `updateBy`
- `updateTime`

### 保存前确认格式

```text
将新增单位：
- 单位编码：CM
- 单位名称：厘米
- 单位类型：长度
- 基准单位：M 米
- 换算率：0.01
是否确认保存？
```

### 失败提示

| 场景 | 给用户的提示 |
| --- | --- |
| 单位编码重复 | 单位编码已存在，请换一个编码，或者编辑已有单位。 |
| 单位类型缺失 | 请说明单位类型，例如长度、数量、面积或重量。 |
| 基准单位不存在 | 没有找到对应基准单位，请先维护基准单位。 |
| 换算率无效 | 换算率必须大于 0，请重新确认。 |
| 无权限 | 当前账号没有维护单位的权限，请联系管理员授权。 |
