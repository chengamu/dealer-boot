# 物料属性接口与数据契约

## 页面与权限

| 项目 | 内容 |
| --- | --- |
| 菜单名称 | 基础信息 / 物料属性 |
| 前端路由 | `/product-master/base-attributes` |
| 前端组件 | `product-center/base/ProductBaseInfoPage.vue` 的 `baseAttribute` 配置 |
| 权限码 | `product:base-attribute:*` |

## 后端接口

| 动作 | 方法 | 路径 | 权限 |
| --- | --- | --- | --- |
| 列表 | GET | `/product-capability/base-attributes/list` | `product:base-attribute:list` |
| 选项 | GET | `/product-capability/base-attributes/options` | `product:base-attribute:list` |
| 详情 | GET | `/product-capability/base-attributes/{id}` | `product:base-attribute:list` |
| 修改前检查 | GET | `/product-capability/base-attributes/{id}/edit-check` | `product:base-attribute:edit` |
| 新增 | POST | `/product-capability/base-attributes` | `product:base-attribute:add` |
| 修改 | PUT | `/product-capability/base-attributes` | `product:base-attribute:edit` |
| 启停 | PUT | `/product-capability/base-attributes/change-status/{id}/{status}` | `product:base-attribute:edit` |
| 删除 | DELETE | `/product-capability/base-attributes/{ids}` | `product:base-attribute:remove` |
| 引用检查 | GET | `/product-capability/base-attributes/{id}/references` | `product:base-attribute:reference` |

## 数据表

| 表名 | 用途 |
| --- | --- |
| `pc_base_attribute` | 定义某个属性分组共享的物料属性字段 |
| `pc_material_attribute` | 保存具体物料的属性值 |
| `pc_material_type_group` | 属性分组来源 |

## 关键字段

| 字段 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| `attribute_id` | bigint | 是 | 内部 ID |
| `attribute_group_code` | varchar | 是 | 属性分组编码 |
| `attribute_group_name_cn` | varchar | 否 | 属性分组中文名称快照 |
| `attribute_code` | varchar | 是 | 稳定属性编码 |
| `attribute_name_cn` | varchar | 是 | 中文名称 |
| `attribute_name_en` | varchar | 否 | 英文名称 |
| `value_type` | varchar | 是 | `TEXT` / `NUMBER` / `BOOLEAN` / `ENUM` |
| `unit_code` | varchar | 否 | 默认单位，仅数字属性使用 |
| `status` | varchar | 是 | `ENABLED` / `DISABLED` |
| `sort_order` | int | 否 | 排序 |
| `remark` | varchar | 否 | 备注 |

## 选择器来源

| 页面字段 | 数据来源 | 过滤条件 |
| --- | --- | --- |
| 属性分组 | `pc_material_type_group` | `status = ENABLED` |
| 值类型 | 前端固定枚举 | `TEXT`、`NUMBER`、`BOOLEAN`、`ENUM` |
| 默认单位 | `pc_unit` | `status = ENABLED`；仅值类型为 `NUMBER` 时显示 |

## 校验规则

- `attribute_group_code`、`attribute_code`、`attribute_name_cn`、`value_type` 必填。
- 属性分组必须来自启用的 `pc_material_type_group`。
- 同一属性分组下 `attribute_code` 不能重复。
- `unit_code` 只能在值类型为 `NUMBER` 时填写。
- 删除前必须检查物料属性值和工程规则引用；有引用时优先停用。
- 启用状态资料不能直接编辑，先停用再编辑。

## 请求示例

```json
{
  "attributeGroupCode": "CONTROL",
  "attributeCode": "CHANNEL_COUNT",
  "attributeNameCn": "通道数",
  "attributeNameEn": "Channel Count",
  "valueType": "NUMBER",
  "unitCode": null,
  "status": "ENABLED",
  "sortOrder": 20,
  "remark": "控制系统共享属性"
}
```

## 响应示例

```json
{
  "attributeId": 10001,
  "attributeGroupCode": "CONTROL",
  "attributeGroupNameCn": "控制系统",
  "attributeCode": "CHANNEL_COUNT",
  "attributeNameCn": "通道数",
  "attributeNameEn": "Channel Count",
  "valueType": "NUMBER",
  "unitCode": null,
  "status": "ENABLED",
  "sortOrder": 20
}
```

## 常见错误

| 错误场景 | 后端提示 | 用户提示 |
| --- | --- | --- |
| 属性分组不存在 | `product.baseAttribute.groupNotFound` | 请选择有效的属性分组。 |
| 属性编码重复 | `product.baseAttribute.codeExists` | 当前分组下属性编码已存在，请换一个稳定编码。 |
| 非数字属性填写单位 | `product.baseAttribute.unitOnlyForNumber` | 只有数字型计量属性才需要默认单位。 |
| 删除已引用属性 | `product.baseAttribute.hasReferences` | 该属性已被物料属性或工程规则使用，请先停用。 |
