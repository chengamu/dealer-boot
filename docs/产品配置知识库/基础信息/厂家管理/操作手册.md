# 厂家管理操作手册

## 功能定位

厂家管理用于维护物料的供货来源和生产来源。系统页面统一叫“厂家”，业务上也可理解为供应商来源。

厂家编号会参与后续物料编码规则的最后三位。物料管理中只选择厂家，并填写厂家物料编码；厂家编号、厂家名称由系统保存为物料快照。

## 页面入口

- 菜单：基础信息 / 厂家管理
- 路由：`/product-master/manufacturers`

## 字段说明

| 字段 | 是否必填 | 说明 |
| --- | --- | --- |
| 厂家编号 | 是 | 3 位业务编号，例如 `001`、`002`、`900` |
| 厂家名称 | 是 | 厂家完整名称 |
| 厂家简称 | 否 | 用于列表和人工识别 |
| 是否厂家 | 否 | 表示该主体可作为生产厂家 |
| 是否供应商 | 否 | 表示该主体可作为供货来源 |
| 联系人 | 否 | 对接人 |
| 联系电话 | 否 | 联系方式 |
| 地址 | 否 | 厂家地址 |
| 排序 | 否 | 列表排序 |
| 备注 | 否 | 补充说明 |

`900` 是系统初始化的通用厂家，用于通用件、暂无厂家或自制件。真实厂家编号由业务人员在厂家管理中维护。

## 常用操作

### 新增厂家

1. 点击新增。
2. 填写厂家编号和厂家名称。
3. 按需填写简称、联系人、联系电话和地址。
4. 保存后默认未审核。
5. 确认资料可用后审核。

### 修改厂家

未审核厂家可以普通修改。已审核厂家如需普通修改，先取消审核，再编辑。

### 删除厂家

已审核厂家不能直接删除。已被物料引用的厂家不能删除，应保留记录并取消审核。

### 引用检查

引用检查会统计当前厂家被物料引用的情况。引用数量大于 0 时不能删除。

## 与物料管理的关系

物料新增或编辑时，厂家字段来自厂家管理。用户选择厂家后，系统保存：

- `manufacturer_id`
- `manufacturer_code`
- `manufacturer_name`

物料表单里另有“厂家物料编码”，用于记录厂家自己的料号，存入 `manufacturer_item_no`。

## 如何确认维护成功

- 厂家能在列表中查到。
- 审核后的厂家能在物料管理的厂家下拉中选择。
- 被物料引用后，厂家引用检查能看到引用数量。

## AI 代办能力

### 支持的意图

```text
意图名称：新增厂家
用户说法：新增厂家 001，名称是某某铝材厂。
```

```text
意图名称：取消审核厂家
用户说法：这个厂家后续不要再给新物料选择。
```

### 必填槽位

- 厂家编号。
- 厂家名称。

### 可推断字段

- 新增厂家默认未审核。
- 未说明是否厂家或供应商时，不自动替用户判断。

### 需要追问

- 厂家编号缺失时必须追问。
- 厂家名称缺失时必须追问。
- 用户要求删除已被物料引用厂家时，应建议取消审核。

### 保存前确认摘要

```text
将新增厂家：
- 厂家编号：001
- 厂家名称：某某铝材厂
- 厂家简称：某某铝材
保存后状态：未审核
是否确认保存？
```
