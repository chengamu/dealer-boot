# 厂家管理接口与数据契约

## 页面与权限

| 项目 | 内容 |
| --- | --- |
| 菜单名称 | 基础信息 / 厂家管理 |
| 前端路由 | `/product-master/manufacturers` |
| 前端组件 | `product-center/base/ProductBaseInfoPage.vue` 的 `manufacturer` 配置 |
| 权限码 | `product:manufacturer:*` |

## 后端接口

| 动作 | 方法 | 路径 | 权限 |
| --- | --- | --- | --- |
| 列表 | GET | `/product-capability/manufacturers/list` | `product:manufacturer:list` |
| options | GET | `/product-capability/manufacturers/options` | `product:manufacturer:list` |
| 详情 | GET | `/product-capability/manufacturers/{id}` | `product:manufacturer:list` |
| 修改前检查 | GET | `/product-capability/manufacturers/{id}/edit-check` | `product:manufacturer:edit` |
| 新增 | POST | `/product-capability/manufacturers` | `product:manufacturer:add` |
| 修改 | PUT | `/product-capability/manufacturers` | `product:manufacturer:edit` |
| 审核 / 取消审核 | PUT | `/product-capability/manufacturers/change-status/{id}/{status}` | `product:manufacturer:changeStatus` |
| 删除 | DELETE | `/product-capability/manufacturers/{ids}` | `product:manufacturer:remove` |
| 引用检查 | GET | `/product-capability/manufacturers/{id}/references` | `product:manufacturer:reference` |
| 导出 | POST | `/product-capability/manufacturers/export` | `product:manufacturer:export` |

## 数据表

| 表名 | 用途 |
| --- | --- |
| `pc_manufacturer` | 厂家主数据 |
| `pc_material` | 物料保存厂家 ID、编号、名称快照和厂家物料编码 |

## 关键字段

| 字段 | 必填 | 说明 |
| --- | --- | --- |
| `manufacturer_id` | 系统生成 | 厂家 ID |
| `tenant_id` | 系统写入 | 租户 ID |
| `manufacturer_code` | 是 | 3 位厂家编号，程序校验唯一 |
| `manufacturer_name` | 是 | 厂家名称 |
| `manufacturer_short_name` | 否 | 厂家简称 |
| `manufacturer_flag` | 否 | 是否厂家 |
| `supplier_flag` | 否 | 是否供应商 |
| `contact_name` | 否 | 联系人 |
| `contact_phone` | 否 | 联系电话 |
| `address` | 否 | 地址 |
| `status` | 是 | `ENABLED` 已审核，`DISABLED` 未审核 |
| `sort_order` | 否 | 排序 |
| `remark` | 否 | 备注 |

## 校验规则

- `manufacturer_code` 必填，且同租户下未删除数据中唯一。
- `manufacturer_name` 必填。
- 新增厂家默认 `status = DISABLED`。
- 已审核厂家不能直接删除。
- 被物料引用的厂家不能删除。
- options 默认返回已审核厂家，用于物料管理厂家下拉。

## 初始化数据

系统只初始化一条必要厂家：

| 厂家编号 | 厂家名称 | 用途 |
| --- | --- | --- |
| `900` | 通用厂家/暂无厂家 | 通用件、暂无厂家、自制件 |

真实厂家不写入初始化脚本，由业务人员维护或通过单独数据脚本导入。

## 可检索性与引用关系

业务编码或单号：

- `manufacturer_code`（厂家编号）

搜索展示字段：

- `manufacturer_code`
- `manufacturer_name`
- `status`
- `updateTime`

主要引用关系：

- 物料管理通过厂家编号引用厂家，并可用于物料编码后缀。

权限过滤依据：

- `tenantId`
- `product:manufacturer:list` / `product:manufacturer:reference`
- `status`
- `delFlag`

详情入口：

| 入口 | 内容 |
| --- | --- |
| 前端路由 | `/product-master/manufacturers` |
| 后端详情接口 | `GET /product-capability/manufacturers/{id}` |

说明：搜索结果只作为入口和摘要展示；厂家状态和引用检查以业务数据库和详情接口为准。

## 常见错误

| 错误场景 | 后端提示 | 用户提示 |
| --- | --- | --- |
| 厂家编号未填 | `product.manufacturer.codeRequired` | 请填写厂家编号。 |
| 厂家名称未填 | `product.manufacturer.nameRequired` | 请填写厂家名称。 |
| 厂家编号重复 | `product.manufacturer.codeExists` | 厂家编号已存在。 |
| 删除已引用厂家 | `product.manufacturer.hasReferences` | 该厂家已被物料引用，不能删除。 |

## AI 工具调用契约

### 意图

```text
意图名称：新增厂家
用户说法：新增厂家 001，名称是某某铝材厂
适用页面：基础信息 / 厂家管理
调用接口：POST /product-capability/manufacturers
权限要求：product:manufacturer:add
```

```text
意图名称：取消审核厂家
用户说法：这个厂家后续不要再给新物料选择
适用页面：基础信息 / 厂家管理
调用接口：PUT /product-capability/manufacturers/change-status/{id}/DISABLED
权限要求：product:manufacturer:changeStatus
```

### 调用接口和权限

| 意图 | 方法 | 路径 | 权限 |
| --- | --- | --- | --- |
| 查询厂家 | GET | `/product-capability/manufacturers/list` | `product:manufacturer:list` |
| 新增厂家 | POST | `/product-capability/manufacturers` | `product:manufacturer:add` |
| 编辑厂家 | GET + PUT | `/product-capability/manufacturers/{id}/edit-check`、`/product-capability/manufacturers` | `product:manufacturer:edit` |
| 审核 / 取消审核 | PUT | `/product-capability/manufacturers/change-status/{id}/{status}` | `product:manufacturer:changeStatus` |
| 查询引用 | GET | `/product-capability/manufacturers/{id}/references` | `product:manufacturer:reference` |

### 必填槽位

- `manufacturerCode`：厂家编号。
- `manufacturerName`：厂家名称。

### 可选槽位

- `manufacturerShortName`：厂家简称。
- `manufacturerFlag`：是否厂家。
- `supplierFlag`：是否供应商。
- `contactName`：联系人。
- `contactPhone`：联系电话。
- `address`：地址。
- `remark`：备注。

### 可推断字段

- 新增厂家默认 `status = DISABLED`，页面展示为未审核。
- 未说明 `manufacturerFlag` 或 `supplierFlag` 时不自动猜测。

### 需要追问

- 厂家编号或厂家名称缺失。
- 用户说的厂家名称无法唯一定位。
- 用户要求编辑已审核厂家，但没有先取消审核。
- 用户要求删除已被物料引用厂家时，应提示不能删除并建议取消审核。

### Payload 边界

- AI 不提交 `tenantId`、`createBy`、`createTime`、`updateBy`、`updateTime`、`delFlag`。
- AI 不让用户填写内部 `manufacturerId`。
- 编辑前必须先调用 `edit-check`；不可编辑时不调用普通 `PUT`。

### 禁止暴露字段

- `manufacturerId`
- `tenantId`
- `createBy`
- `createTime`
- `updateBy`
- `updateTime`
- `delFlag`

### 保存前确认格式

```text
将新增厂家：
- 厂家编号：001
- 厂家名称：某某铝材厂
- 厂家简称：某某铝材
保存后状态：未审核
是否确认保存？
```

### 失败提示

| 场景 | 用户提示 |
| --- | --- |
| 厂家编号缺失 | 请提供 3 位厂家编号。 |
| 厂家名称缺失 | 请提供厂家名称。 |
| 厂家编号重复 | 厂家编号已存在，请换一个编号或编辑已有厂家。 |
| 已审核不可普通编辑 | 该厂家已审核，请先取消审核后再编辑。 |
| 删除已引用厂家 | 该厂家已被物料引用，不能删除；可以取消审核让它不再进入新物料选择。 |
