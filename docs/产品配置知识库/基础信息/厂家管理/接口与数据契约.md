# 厂家管理接口与数据契约

## 页面与权限

| 项目 | 内容 |
| --- | --- |
| 菜单名称 | 基础信息 / 厂家管理 |
| 前端路由 | `/product-master/manufacturers` |
| 前端组件 | `product-center/base/ProductBaseInfoPage.vue` 的 `manufacturer` 配置 |
| 权限码 | `product:manufacturer:*` |

## 后端接口

| 动作 | 方法 | 路径 | 权限 |
| --- | --- | --- | --- |
| 列表 | GET | `/product-capability/manufacturers/list` | `product:manufacturer:list` |
| options | GET | `/product-capability/manufacturers/options` | `product:manufacturer:list` |
| 详情 | GET | `/product-capability/manufacturers/{id}` | `product:manufacturer:list` |
| 修改前检查 | GET | `/product-capability/manufacturers/{id}/edit-check` | `product:manufacturer:edit` |
| 新增 | POST | `/product-capability/manufacturers` | `product:manufacturer:add` |
| 修改 | PUT | `/product-capability/manufacturers` | `product:manufacturer:edit` |
| 审核 / 取消审核 | PUT | `/product-capability/manufacturers/change-status/{id}/{status}` | `product:manufacturer:changeStatus` |
| 删除 | DELETE | `/product-capability/manufacturers/{ids}` | `product:manufacturer:remove` |
| 引用检查 | GET | `/product-capability/manufacturers/{id}/references` | `product:manufacturer:reference` |
| 导出 | POST | `/product-capability/manufacturers/export` | `product:manufacturer:export` |

## 数据表

| 表名 | 用途 |
| --- | --- |
| `pc_manufacturer` | 厂家主数据 |
| `pc_material` | 物料保存厂家 ID、编号、名称快照和厂家物料编码 |

## 关键字段

| 字段 | 必填 | 说明 |
| --- | --- | --- |
| `manufacturer_id` | 系统生成 | 厂家 ID |
| `tenant_id` | 系统写入 | 租户 ID |
| `manufacturer_code` | 是 | 3 位厂家编号，程序校验唯一 |
| `manufacturer_name` | 是 | 厂家名称 |
| `manufacturer_short_name` | 否 | 厂家简称 |
| `manufacturer_flag` | 否 | 是否厂家 |
| `supplier_flag` | 否 | 是否供应商 |
| `contact_name` | 否 | 联系人 |
| `contact_phone` | 否 | 联系电话 |
| `address` | 否 | 地址 |
| `status` | 是 | `ENABLED` 已审核，`DISABLED` 未审核 |
| `sort_order` | 否 | 排序 |
| `remark` | 否 | 备注 |

## 校验规则

- `manufacturer_code` 必填，且同租户下未删除数据中唯一。
- `manufacturer_name` 必填。
- 新增厂家默认 `status = DISABLED`。
- 已审核厂家不能直接删除。
- 被物料引用的厂家不能删除。
- options 默认返回已审核厂家，用于物料管理厂家下拉。

## 初始化数据

系统只初始化一条必要厂家：

| 厂家编号 | 厂家名称 | 用途 |
| --- | --- | --- |
| `900` | 通用厂家/暂无厂家 | 通用件、暂无厂家、自制件 |

真实厂家不写入初始化脚本，由业务人员维护或通过单独数据脚本导入。

## 常见错误

| 错误场景 | 后端提示 | 用户提示 |
| --- | --- | --- |
| 厂家编号未填 | `product.manufacturer.codeRequired` | 请填写厂家编号。 |
| 厂家名称未填 | `product.manufacturer.nameRequired` | 请填写厂家名称。 |
| 厂家编号重复 | `product.manufacturer.codeExists` | 厂家编号已存在。 |
| 删除已引用厂家 | `product.manufacturer.hasReferences` | 该厂家已被物料引用，不能删除。 |
