# 组件包接口与数据契约

## 页面与权限

| 项目 | 内容 |
| --- | --- |
| 菜单名称 | 基础信息 / 组件包 |
| 前端路由 | `/product-master/components` |
| 前端组件 | `admin-ui/src/pages/product-center/base/ProductBaseInfoPage.vue` 的 `component` 配置 |
| 前端 API | `admin-ui/src/api/product-capability/component.ts` |
| Controller | `ProductComponentController` |
| Service | `ProductComponentService` |
| 权限码 | `product:base:list`、`product:base:add`、`product:base:edit`、`product:base:remove`、`product:base:reference` |

## 后端接口

| 动作 | 方法 | 路径 | 权限 |
| --- | --- | --- | --- |
| 列表 | GET | `/product-capability/components/list` | `product:base:list` |
| 选项 | GET | `/product-capability/components/options` | `product:base:list` |
| 详情 | GET | `/product-capability/components/{id}` | `product:base:list` |
| 修改前检查 | GET | `/product-capability/components/{id}/edit-check` | `product:base:edit` |
| 新增 | POST | `/product-capability/components` | `product:base:add` |
| 修改 | PUT | `/product-capability/components` | `product:base:edit` |
| 启停 | PUT | `/product-capability/components/change-status/{id}/{status}` | `product:base:edit` |
| 删除 | DELETE | `/product-capability/components/{ids}` | `product:base:remove` |
| 引用检查 | GET | `/product-capability/components/{id}/references` | `product:base:reference` |

## 数据表

| 表名 | 用途 |
| --- | --- |
| `pc_component` | 组件包主表，只保存组件包自身字段 |
| `pc_component_item` | 组件包和物料的组成关系 |
| `pc_media_binding` | 组件包资料绑定 |

## 关键字段

| 字段 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| `component_id` | bigint | 系统生成 | 组件包内部 ID |
| `component_code` | varchar | 是 | 组件编码 |
| `component_name_cn` | varchar | 是 | 组件中文名称 |
| `component_name_en` | varchar | 否 | 组件英文名称 |
| `component_type` | varchar | 否 | 组件类型 |
| `business_type` | varchar | 否 | 业务类型 |
| `default_qty` | numeric | 否 | 默认数量 |
| `qty_mode` | varchar | 否 | 数量模式 |
| `unit_code` | varchar | 否 | 单位编码 |
| `status` | varchar | 是 | `ENABLED` / `DISABLED` |

## 校验规则

- 组件包新增和修改必须提供组件编码、中文名称。
- 组件包主表不保存默认物料；物料组成必须通过 `pc_component_item` 维护。
- 删除前应检查组件明细和资料绑定引用。

## AI 工具调用契约

AI 只调用组件包接口和组件明细接口，不直接写数据库。新增组件包后，如用户同时给出物料组成，应继续调用组件明细新增接口逐行保存。
