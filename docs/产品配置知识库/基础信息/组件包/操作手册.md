# 组件包操作手册

## 功能定位

组件包用于维护可复用的组件组合主档，例如电机系统包、安装配件包、底杆包等。

组件包主表只描述“包”本身，不保存默认物料。组件包由哪些物料组成，统一在组件明细中维护。

```text
基础字典 / 单位管理 / 物料管理 -> 组件包 -> 组件明细 -> 产品工程设计 -> 产品配置中心
```

## 使用前准备

- 基础字典：组件类型、业务类型、工程数量模式。
- 单位管理：组件包默认数量单位。
- 物料管理：组件明细后续要引用的物料先维护好。

## 页面入口

- 菜单：基础信息 / 组件包。
- 路由：`/product-master/components`。

## 字段说明

| 字段 | 是否必填 | 说明 |
| --- | --- | --- |
| 组件编码 | 是 | 组件包稳定业务编码。 |
| 组件中文名称 | 是 | 业务识别名称。 |
| 组件英文名称 | 否 | 英文环境展示。 |
| 组件类型 | 否 | 来自基础字典 `product_component_type`。 |
| 业务类型 | 否 | 来自基础字典 `product_business_type`。 |
| 默认数量 | 否 | 组件包作为整体被引用时的默认数量。 |
| 数量模式 | 否 | 来自基础字典 `engineering_qty_mode`。 |
| 单位编码 | 否 | 来自单位管理。 |
| 状态 | 是 | `ENABLED` / `DISABLED`。 |

## 常用操作

1. 新增组件包，填写组件编码、名称、类型和默认数量口径。
2. 保存后在组件包行点击“管理明细”。
3. 在组件明细中绑定物料、数量、单位和是否必需。

## 删除或停用

组件包已有组件明细或资料绑定时，不建议删除，应优先停用。
