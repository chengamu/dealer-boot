# 组件明细操作手册

## 功能定位

组件明细用于维护一个组件包由哪些物料组成，以及默认用量、单位和是否必需。

```text
产品字典 / 单位管理 / 物料资料 -> 组件包 -> 组件明细 -> 产品工程设计 -> 产品配置中心
```

它解决的是“组件包下面包含什么物料”的基础资料问题，不负责尺寸可做性、动态带出规则或生产用料计算。

## 使用对象

- 主要使用人：技术科、产品资料维护人员。
- 辅助使用人：测试人员、实施人员。
- 不建议使用人：只查看销售配置结果的业务人员。

## 使用前准备

使用本功能前，需要先维护：

- 产品字典：组件包类型、业务类型、工程数量模式。
- 单位管理：组件包和明细用量需要的单位。
- 物料资料：组件明细引用的物料必须先存在。
- 组件包：组件明细必须归属到一个组件包。

## 推荐录入顺序

1. 在物料资料中维护组件包会用到的物料。
2. 在组件包中维护组件编码、名称、类型、默认数量和单位。
3. 进入组件明细，选择组件包和物料，维护默认数量、数量公式、单位、是否必需。
4. 启用组件明细后，再进入工程配置或产品配置引用。

## 页面入口

- 菜单：基础信息 / 组件明细，或在组件包列表点击“管理明细”。
- 路由：`/product-master/component-items`，也可通过 `/product-master/components?tab=componentItem` 打开。

## 字段说明

| 字段 | 是否必填 | 怎么填写 | 注意事项 |
| --- | --- | --- | --- |
| 组件编码 | 是 | 从已启用组件包中选择 | 从组件包行进入时会带入当前组件编码。 |
| 物料编码 | 是 | 从物料资料中选择 | 选择后会带出物料中文名称和工程使用单位。 |
| 物料中文名称 | 否 | 系统带出 | 用于识别明细，不建议手填覆盖。 |
| 数量公式 | 否 | 填写简单用量表达式 | 当前仅保存文本，工程计算规则不在本页面定义。 |
| 默认数量 | 否 | 填写固定默认用量 | 批量录入默认值为 1。 |
| 单位编码 | 否 | 从单位管理选择或由物料带出 | 建议与物料工程使用单位一致。 |
| 是否必需 | 否 | 是 / 否 | 必需表示组件包默认包含该物料。 |
| 排序 | 否 | 数字越小越靠前 | 批量录入按 10、20、30 递增。 |
| 状态 | 是 | 启用 / 停用 | 停用后不应再进入新增业务选择器。 |

## 常用操作

### 新增

1. 打开组件明细页面。
2. 选择组件编码。
3. 选择物料编码。
4. 填写默认数量、单位、是否必需和排序。
5. 保存。

### 批量录入

组件明细页面提供批量录入抽屉，可以一次维护多行明细。

当前批量录入是前端批量编辑体验，保存时逐行调用新增或修改接口，不是单独的后端批量保存接口。

### 编辑

点击修改前，页面会调用修改前检查。启用状态的基础资料不允许普通编辑，需要先停用再编辑。

### 删除或停用

如果组件明细后续已被工程配置、订单快照或发布包引用，原则上应优先停用，不建议删除。当前引用检查能力仍需人工测试确认。

### 查询

推荐按组件编码、物料编码、状态查询。组件包行进入明细时，会优先按当前组件编码过滤。

## 常见问题

| 问题 | 原因 | 处理方式 |
| --- | --- | --- |
| 找不到物料 | 物料未维护或未启用 | 先到物料资料维护并启用。 |
| 编辑按钮打不开 | 当前明细已启用 | 先停用，再进行普通编辑。 |
| 批量保存部分失败 | 当前是逐行保存 | 根据失败提示修正对应行后重新保存；需要人工测试确认失败行表现。 |

## 如何确认维护成功

- 组件明细列表能看到新维护的物料行。
- 从组件包点击“管理明细”时，能看到该组件包下的明细。
- 启用的组件明细后续能被工程配置或配置检查正确引用。

## AI 代办能力

### 支持的意图

```text
意图名称：新增组件明细
用户说法：给电机系统包加一个电机物料，默认数量 1 个，必需。
```

### 调用入口

- 页面：`/product-master/component-items`
- 接口：`POST /product-capability/component-items`
- 权限：`product:component-item:add`

### 必填槽位

- 组件编码
- 物料编码

### 可选槽位

- 数量公式
- 默认数量
- 单位编码
- 是否必需
- 排序
- 状态

### 可推断字段

- 状态默认启用。
- 默认数量未说明时可填 1。
- 单位编码可从物料资料的工程使用单位带出。

### 需要追问

- 缺少组件编码时必须追问。
- 缺少物料编码时必须追问。
- 物料选择存在多个匹配项时必须追问。

### 失败提示

| 场景 | 给用户的提示 |
| --- | --- |
| 组件不存在 | 请先维护组件包，或提供已有组件编码。 |
| 物料不存在 | 请先维护物料资料，或提供已有物料编码。 |
| 当前明细不可编辑 | 当前组件明细已启用，请先停用后再编辑。 |

### 保存前确认摘要

```text
将为组件包【组件编码】新增明细：
- 物料：物料编码
- 默认数量：数量 + 单位
- 是否必需：是/否
确认后保存。
```
