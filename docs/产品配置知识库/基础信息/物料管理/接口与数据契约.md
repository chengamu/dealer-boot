# 物料管理接口与数据契约

## 页面与权限

| 项目 | 内容 |
| --- | --- |
| 菜单名称 | 基础信息 / 物料管理 |
| 前端路由 | `/product-master/materials` |
| 前端组件 | `product-center/base/ProductBaseInfoPage.vue` 的 `material` 配置 |
| 权限码 | `product:base:*` |

## 后端接口

| 动作 | 方法 | 路径 | 权限 |
| --- | --- | --- | --- |
| 列表 | GET | `/product-capability/materials/list` | `product:base:list` |
| options | GET | `/product-capability/materials/options` | `product:base:list` |
| 详情 | GET | `/product-capability/materials/{id}` | `product:base:list` |
| 修改前检查 | GET | `/product-capability/materials/{id}/edit-check` | `product:base:edit` |
| 新增 | POST | `/product-capability/materials` | `product:base:add` |
| 修改 | PUT | `/product-capability/materials` | `product:base:edit` |
| 超级修改 | PUT | `/product-capability/materials/super-update` | `product:base:superEdit` |
| 启停 | PUT | `/product-capability/materials/change-status/{id}/{status}` | `product:base:edit` |
| 删除 | DELETE | `/product-capability/materials/{ids}` | `product:base:remove` |
| 引用检查 | GET | `/product-capability/materials/{id}/references` | `product:base:reference` |

物料属性值有独立接口和隐藏台账，普通维护入口在物料管理新增 / 编辑抽屉里同步保存。

## 数据表

| 表名 | 用途 |
| --- | --- |
| `pc_material` | 统一物料主档 |
| `pc_material_attribute` | 具体物料的属性值 |
| `pc_material_type` | 物料类型来源 |
| `pc_material_type_group` | 属性分组来源 |
| `pc_base_attribute` | 属性模板来源 |
| `pc_unit` | 单位来源 |

| 字段 | 类型 | 必填 | 说明 |

| `material_id` | bigint | 系统生成 | 内部 ID |
| `material_code` | varchar | 是 | 物料编码 |
| `material_name_cn` | varchar | 是 | 物料中文名称 |
| `material_name_en` | varchar | 否 | 物料英文名称 |
| `material_type_id` | bigint | 是 | 物料类型 ID 快照 |
| `material_type_code` | varchar | 是 | 物料类型编码快照 |
| `material_type_name_cn` | varchar | 否 | 物料类型中文名称快照 |
| `attribute_group_id` | bigint | 是 | 属性分组 ID 快照 |
| `attribute_group_code` | varchar | 是 | 属性分组编码快照 |
| `attribute_group_name_cn` | varchar | 否 | 属性分组中文名称快照 |
| `fabric_series_id` | bigint | 面料必填 | 面料系列 ID |
| `fabric_series_code` | varchar | 面料必填 | 面料系列编码快照 |
| `fabric_series_name_cn` | varchar | 否 | 面料系列中文名称快照 |
| `business_type` | varchar | 否 | 业务类型 |
| `unit_code` | varchar | 否 | 主单位 |
| `purchase_unit_code` | varchar | 否 | 采购单位 |
| `inventory_unit_code` | varchar | 否 | 库存单位 |
| `usage_unit_code` | varchar | 否 | 工程使用单位 |

## 选择器来源

| 页面字段 | 数据来源 | 过滤条件 |
| --- | --- | --- |
| 物料类型 | `pc_material_type` | `status = ENABLED` |
| 面料系列 | `pc_fabric_series` | `status = ENABLED`；仅物料类型编码为 `FABRIC` 时显示 |
| 业务类型 | 基础字典 `product_business_type` | 启用项 |
| 主单位 / 采购单位 / 库存单位 / 工程使用单位 | `pc_unit` | 启用项 |
| 类型属性定义 | `pc_base_attribute` | `status = ENABLED`，`attribute_group_code = 当前物料类型所属属性分组` |
| 类型属性值单位 | `pc_unit` | 启用项；通常仅数字属性显示 |

## 校验规则

- `material_code`、`material_name_cn`、`material_type_code` 必填。
- 物料类型必须来自启用的 `pc_material_type`。
- 保存物料时，后端根据物料类型回填 `material_type_*` 和 `attribute_group_*` 快照字段。
- `material_type_code = FABRIC` 时必须选择面料系列。
- `material_type_code != FABRIC` 时后端清空面料系列快照字段。
- `material_code` 在同一租户下由接口服务层校验唯一。
- 相同物料中文名称、主规格、供应商编码或供应商名称的物料由接口服务层判重。
- 保存物料时如果带 `attributeList`，后端同步保存到 `pc_material_attribute`。
- 删除物料前应检查引用；有引用时优先停用。
- 启用状态资料不能直接编辑，先停用再编辑；超级修改走独立权限和独立接口。

## 请求示例

```json
{
  "materialCode": "MOTOR_AOK_45_ZIGBEE",
  "materialNameCn": "AOK 45 管状电机 Zigbee",
  "materialNameEn": "AOK 45 Tubular Motor Zigbee",
  "materialTypeCode": "MOTOR",
  "businessType": "PURCHASE",
  "unitCode": "PCS",
  "supplierName": "电机供应商",
  "primarySpec": "45mm Zigbee",
  "status": "ENABLED",
  "attributeList": [
    {
      "attributeCode": "VOLTAGE",
      "attributeNameCn": "电压",
      "valueText": "220V",
      "valueNumber": 220,
      "status": "ENABLED"
    }
  ]
}
```

```json
  "status": "ENABLED"
}
```

## 常见错误

| 错误场景 | 后端提示 | 用户提示 |
| --- | --- | --- |
| 物料类型未填 | `product.material.materialTypeRequired` | 请选择物料类型。 |
| 物料类型不存在 | `product.material.materialTypeNotFound` | 请选择有效的物料类型。 |
| 面料未选系列 | `product.material.fabricSeriesRequired` | 面料物料必须选择面料系列。 |
| 物料编码重复 | `product.material.codeExists` | 物料编码已存在，请换一个编码或编辑已有物料。 |
| 删除已引用物料 | `product.material.hasReferences` | 该物料已被引用，请先停用。 |
