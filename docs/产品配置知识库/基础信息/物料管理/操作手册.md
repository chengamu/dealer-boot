# 物料管理操作手册

## 功能定位

物料管理用于维护后续工程配置、组件配方、报价、采购、库存和订单快照都会引用的统一物料主档。

这里维护的是“可以被系统识别和复用的物料”，不是销售产品，也不是工程方案。面料、铝材、下杆、轨道、电机、拉珠、遥控器、太阳能板、配件、安装件、包装件、标准成品，都应该进入物料管理。

当前基础信息顺序：

```text
基础字典 -> 单位管理 -> 物料类型 -> 物料属性 -> 物料管理
```

## 使用前准备

使用物料管理前，需要先准备：

- 单位管理：主单位、副单位要能选择。
- 物料类型：维护属性分组以及分组下的物料类型。
- 物料属性：为属性分组准备需要填写的扩展参数。
- 基础字典：维护简单枚举。

物料类型决定当前物料加载哪一组类型属性。例如电机、遥控器、拉珠可以归到控制系统分组，共用电压、协议、通道数等属性模板。

## 页面入口

- 菜单：基础信息 / 物料管理
- 路由：`/product-master/materials`

## 物料状态

物料只有一个业务状态：

| 状态 | 含义 | 是否可编辑 | 是否进入新增业务选择 |
| --- | --- | --- | --- |
| 未审核 | 新增或取消审核后的草稿资料 | 可以普通修改 | 不进入 |
| 已审核 | 已确认可用的物料 | 不可普通修改 | 进入 |

审核会记录最后一次审核人和审核日期。取消审核只改变当前可用状态，不清空最后审核记录。

## 字段说明

| 字段 | 是否必填 | 怎么填写 | 注意事项 |
| --- | --- | --- | --- |
| 物料编码 | 是 | 稳定业务编码或 ERP 编码，例如 `H0003430` | 全局唯一，被后续业务引用 |
| 物料名称 | 是 | 中文识别名称，例如 `38管` | 给技术、采购和工程配置识别 |
| 物料英文名称 | 否 | 英文名称 | 用于英文环境 |
| 物料类型 | 是 | 从物料类型选择 | 系统自动带出属性分组和类型属性 |
| 型号 | 否 | 型号或款号，例如 `HC-IV-25` | 结构化字段，可为空 |
| 规格 | 是 | 规格描述，例如 `25mm cell / 0.45mm / 2.5m` | 唯一性判断会用到 |
| 颜色 | 否 | 颜色名称 | 常用但不强制 |
| 克重/重量 | 否 | 数值 | 面料、铝材等需要时填写 |
| 主单位 | 是 | 从单位管理选择 | 业务展示和使用的主单位 |
| 副单位 | 否 | 从单位管理选择 | 用于展示另一种常用口径，不处理库存换算 |
| 单价 | 否 | 当前物料的业务单价 | 不是最终销售价格 |
| 类型属性 | 否 | 由物料类型自动加载 | 随物料一起保存 |
| 厂家名称 | 否 | 生产厂家名称 | 页面优先展示名称 |
| 厂家编码 | 否 | 厂家业务编码 | 保留字段，可低频维护 |
| 厂家料号 | 否 | 厂家自己的物料号 | 采购沟通常用 |
| 备注 | 否 | 补充说明 | 不写临时沟通内容 |

`规格型号` 是系统展示文本，不给用户手工录入。保存时后端根据型号和规格自动生成：

- 有型号和规格：`型号：{model}；规格：{spec}`
- 只有规格：使用规格

## 常用操作

### 新增物料

1. 点击新增。
2. 填写物料编码、物料名称、物料类型。
3. 填写型号、规格、颜色、克重/重量。
4. 选择主单位，可选副单位。
5. 按物料类型填写类型属性。
6. 如有厂家信息，填写厂家名称、厂家编码、厂家料号。
7. 保存后物料处于未审核状态。
8. 确认无误后点击审核。

### 编辑物料

未审核物料可以普通编辑。

已审核物料需要先取消审核，再普通编辑。受控修正可以使用超级修改，超级修改有独立权限和独立接口。

新增、编辑和超级修改抽屉有未保存内容保护。点击关闭、取消、遮罩或按 Esc 时，如内容已变更会先弹出确认提示。

### 审核物料

审核表示物料已经确认可用，可以进入组件、工程、报价等后续业务选择。

审核后系统记录审核人和审核日期。

### 取消审核物料

取消审核表示物料不再给新增业务选择。历史引用继续保留，不影响已经保存的订单快照或资料绑定。

被引用物料不能删除，但通常可以取消审核。只有存在明确运行中阻断时，后端才应拒绝取消审核并说明原因。

### 删除物料

删除前必须做引用检查。

如果物料已经被物料属性值、资料绑定、工程规则或订单快照引用，不允许删除，应改为取消审核。

## 查询物料

推荐高频查询条件：

- 物料编码
- 物料名称
- 物料类型
- 型号
- 规格
- 审核状态

物料类型使用可搜索下拉，页面展示类型名称。`规格型号` 是展示文本，不作为默认查询条件。

## 对后续业务的影响

物料管理会影响：

- 工程配方：后续工程结构会引用物料。
- 产品工程设计：工程构成项、可选范围、能力规则、带出规则引用物料。
- 产品配置中心：配置答案后续会间接引用物料或组件。
- 报价：读取物料编码、名称、型号、规格、单位和单价。
- 采购和库存：读取厂家、厂家料号、单位和物料状态。
- 订单快照：保存物料编码、名称、型号、规格等快照，避免后续资料变动影响历史订单。
- AI 代办：根据物料类型追问对应参数。

## 常见问题

| 问题 | 原因 | 处理方式 |
| --- | --- | --- |
| 物料类型下拉没有想要的值 | 物料类型没有维护或未审核/未启用 | 先到物料类型维护分组和类型 |
| 类型属性没有出现 | 当前物料类型所属属性分组没有维护物料属性 | 先到物料属性维护该分组的属性字段 |
| 已审核物料不能编辑 | 已审核表示可进入业务选择 | 先取消审核，再普通编辑 |
| 被引用物料不能删除 | 删除会破坏历史关系 | 取消审核，让它不再进入新增业务 |
| 规格型号在哪里填 | 不需要填 | 系统按型号和规格自动生成 |
| 批量导入在哪里 | 当前不做批量导入 | 后续接入 AI 表格整理后再做录入草稿 |

## 如何确认维护成功

- 物料能在列表中查到。
- 列表能看到物料编码、名称、物料类型、型号、规格、单位、副单位、单价、审核人、审核日期、编制人、编制日期和厂家。
- 详情或编辑时能看到当前物料类型对应的类型属性值。
- 未审核物料可以编辑，审核后进入业务选择。
- 取消审核后不再进入新增业务选择。

### 必填槽位

- 物料编码。
- 物料名称。
- 物料类型。
- 规格。
- 主单位。

### 需要追问

- 用户没有说明物料类型时，必须追问。
- 用户没有说明规格时，必须追问。
- 当前物料类型对应的关键类型属性缺失时，需要追问。
- 用户提供厂家料号但没有厂家名称时，可以先保存料号，也可以追问厂家。
- 用户要求删除已被引用物料时，需要提示风险并建议取消审核。
