# 基础属性接口与数据契约

## 页面与权限

| 项目 | 内容 |
| --- | --- |
| 菜单名称 | 基础信息 / 基础属性 |
| 前端路由 | `/product-master/base-attributes` |
| 前端组件 | `product-center/base/ProductBaseInfoPage.vue` 的 `baseAttribute` 配置 |
| 权限码 | `product:baseAttribute:*` |

## 后端接口

| 动作 | 方法 | 路径 | 权限 |
| --- | --- | --- | --- |
| 列表 | GET | `/product-capability/base-attributes/list` | `product:baseAttribute:list` |
| 选项 | GET | `/product-capability/base-attributes/options` | `product:baseAttribute:list` |
| 详情 | GET | `/product-capability/base-attributes/{id}` | `product:baseAttribute:query` |
| 新增 | POST | `/product-capability/base-attributes` | `product:baseAttribute:add` |
| 修改 | PUT | `/product-capability/base-attributes` | `product:baseAttribute:edit` |
| 启停 | PUT | `/product-capability/base-attributes/change-status/{id}/{status}` | `product:baseAttribute:edit` |
| 删除 | DELETE | `/product-capability/base-attributes/{ids}` | `product:baseAttribute:remove` |
| 引用检查 | GET | `/product-capability/base-attributes/{id}/references` | `product:baseAttribute:reference` |

## 数据表

| 表名 | 用途 |
| --- | --- |
| `pc_base_attribute` | 定义物料属性字段，例如厚度、门幅、颜色、通道数。 |
| `pc_material_attribute` | 给具体物料保存属性值，引用 `pc_base_attribute.attribute_id`。 |

## 关键字段

| 字段 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| `attribute_id` | bigint | 是 | 内部 ID，不在普通录入界面展示。 |
| `attribute_group` | varchar | 是 | 属性分组。页面应使用产品字典选择器，不应手填。 |
| `attribute_code` | varchar | 是 | 稳定属性编码。 |
| `attribute_name_cn` | varchar | 是 | 中文名称。 |
| `attribute_name_en` | varchar | 否 | 英文名称。 |
| `value_type` | varchar | 是 | 值类型，决定物料属性录入控件。 |
| `unit_code` | varchar | 否 | 默认单位，仅适用于数字型计量属性。 |
| `material_types` | varchar | 是 | 适用物料类型，建议保存为稳定编码集合。 |
| `extra_json` | text | 否 | 扩展配置。普通 UI 不应直接暴露 JSON。 |
| `status` | varchar | 是 | 启用或停用。 |
| `sort_order` | int | 否 | 排序。 |
| `remark` | varchar | 否 | 备注。 |

## 选择器来源

| 页面字段 | 数据来源 | 过滤条件 |
| --- | --- | --- |
| 属性分组 | 产品字典 `product_attribute_group` | 启用项 |
| 值类型 | 产品字典 `product_attribute_value_type` | 启用项 |
| 默认单位 | `pc_unit` | 启用项；值类型为数字时显示；按属性语义过滤单位类型 |
| 适用物料类型 | 产品字典 `product_material_type` | 启用项；支持多选 |

## 状态和字典

| 字段 | 来源 | 可选值 |
| --- | --- | --- |
| `attribute_group` | `pc_product_dict_item` | `FABRIC`、`CONTROL`、`SIZE` 等 |
| `value_type` | `pc_product_dict_item` | `TEXT`、`NUMBER`、`BOOLEAN`、`OPTION` |
| `material_types` | `pc_product_dict_item` | `FABRIC`、`MOTOR`、`REMOTE` 等 |
| `unit_code` | `pc_unit` | `MM`、`CM`、`M`、`KG` 等 |

## 校验规则

- 同一租户下 `attribute_code` 不能重复。
- `attribute_group` 必须来自启用产品字典。
- `value_type` 必须来自启用产品字典。
- `material_types` 必须来自启用物料类型字典。
- `unit_code` 只能在值类型为数字时填写。
- 文本、开关、选项属性不允许保存默认单位。
- 删除前必须检查 `pc_material_attribute` 是否存在引用。
- 已被工程规则引用的属性不允许直接删除。

## 请求示例

```json
{
  "attributeGroup": "CONTROL",
  "attributeCode": "CHANNEL_COUNT",
  "attributeNameCn": "通道数",
  "attributeNameEn": "Channel Count",
  "valueType": "NUMBER",
  "unitCode": null,
  "materialTypes": "REMOTE",
  "status": "ENABLED",
  "sortOrder": 20,
  "remark": "遥控器通道属性"
}
```

## 常见错误

| 错误场景 | 后端提示 | 用户提示 |
| --- | --- | --- |
| 属性编码重复 | `product.baseAttribute.codeExists` | 属性编码已存在，请换一个稳定编码。 |
| 非数字属性填写单位 | `product.baseAttribute.unitOnlyForNumber` | 只有数字型计量属性才需要默认单位。 |
| 物料类型不存在 | `product.baseAttribute.materialTypeInvalid` | 请选择有效的适用物料类型。 |
| 删除已引用属性 | `product.baseAttribute.hasReferences` | 该属性已被物料属性或工程规则使用，请先停用。 |
