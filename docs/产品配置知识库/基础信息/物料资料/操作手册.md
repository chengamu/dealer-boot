# 物料资料操作手册

## 功能定位

物料资料用于维护后续工程配置、组件包、面料 SKU、采购、库存和订单快照都会引用的统一物料主档。

这里维护的是“可以被系统识别和复用的物料”，不是销售产品，也不是工程方案。面料、铝材、下杆、轨道、电机、拉珠、遥控器、太阳能板、配件、安装件、包装件、标准成品，都应该先进入物料资料。

业务顺序：

```text
产品字典 / 单位管理 / 物料属性 -> 面料系列 -> 物料资料 -> 组件包 / 组件明细 -> 产品工程设计 -> 产品配置中心
```

## 使用对象

- 主要使用人：技术科、基础资料维护人员。
- 辅助使用人：采购、实施人员、测试人员。
- 不建议使用人：普通销售、客户。

## 这个功能维护什么

物料资料维护的是一张统一物料主表，例如：

```text
面料：200CM 白色涂层布
铝材：6米/支 双银色圆管
系统：38MM 手托阻力调节盘弹簧棒
配件：安装码、螺丝、尾塞
包装：纸箱、PET盒、包装条
标准成品：55寸五拉蜂巢帘
```

不同物料类型会有不同参数。比如面料关注门幅、颜色、厚度；电机关注电压、协议、通道；铝材关注规格、颜色、厚度。参数字段从“物料属性”维护，物料资料只填写具体值。

## 使用前准备

使用本功能前，需要先维护：

- 产品字典：至少包含物料类型、业务类型、物料属性分组。
- 单位管理：采购单位、库存单位、工程使用单位要能选择。
- 物料属性：为面料、电机、遥控器、铝材、配件等类型准备可填写的参数字段。
- 面料系列：新增 `FABRIC` 物料前必须先准备系列主数据。

如果这些基础资料没有准备好，物料资料仍可录入基础字段，但类型属性、单位和后续工程配置会不完整。

## 推荐录入顺序

1. 先确认物料类型是否存在，例如面料、铝材、电机、遥控器、配件、安装件、包装件。
2. 先确认单位是否存在，例如米、厘米、个、套、支、卷、平方米、千克。
3. 先确认物料属性是否覆盖当前物料类型，例如颜色、门幅、厚度、电压、协议。
4. 如果录入面料，先确认面料系列已存在。
5. 新增物料资料，先填基础信息和主规格。
6. `FABRIC` 物料必须选择面料系列。
7. 根据物料类型填写类型属性。
8. 如采购会用，补供应商、供应商料号、采购单价、成本单价。
9. 保存后在列表中检查物料编码、名称、类型、面料系列、规格摘要、供应商和价格是否正确。

## 页面入口

- 菜单：基础信息 / 物料资料
- 路由：`/product-master/materials`

## 字段说明

| 字段 | 是否必填 | 怎么填写 | 注意事项 |
| --- | --- | --- | --- |
| 物料编码 | 是 | 稳定业务编码或 ERP 编码，例如 `H0003430` | 保存后会被组件包、工程方案、订单快照引用，不建议随意修改 |
| 物料中文名称 | 是 | 中文识别名称，例如 `6米/支 双银色圆管` | 给技术科、采购和工程配置识别 |
| 物料英文名称 | 否 | 英文名称 | 用于英文环境 |
| 物料类型 | 是 | 从产品字典 `product_material_type` 选择 | 选择后会决定类型属性区显示哪些字段 |
| 面料系列 | `FABRIC` 必填 | 从面料系列选择 | 仅物料类型为 `FABRIC` 时显示；非面料保存时系统清空 |
| 业务类型 | 否 | 从产品字典 `product_business_type` 选择 | 用于区分采购件、成品、服务等业务口径 |
| 主单位 | 否 | 从单位管理选择 | 通用展示单位，采购、库存、工程可以另设 |
| 采购单位 | 否 | 从单位管理选择 | 采购报价和采购下单使用 |
| 库存单位 | 否 | 从单位管理选择 | 后续库存台账使用 |
| 工程使用单位 | 否 | 从单位管理选择 | 工程配置、组件包、带出规则使用 |
| 供应商编码 | 否 | 供应商业务编码 | 后续采购对账和供应商匹配使用 |
| 供应商名称 | 否 | 供应商名称 | 列表会展示，方便采购查看 |
| 供应商料号 | 否 | 供应商自己的物料号 | 采购经常会按这个编号沟通 |
| 标准规格 | 否 | 物料主规格，例如 `200CM`、`6米/支 双` | 用于生成规格摘要 |
| 规格摘要 | 否 | 可读规格文本 | 可由主规格和类型属性生成，也允许人工微调 |
| 主颜色 | 否 | 颜色名称 | 面料、铝材、配件常用 |
| 主重量 | 否 | 重量数值 | 只有业务需要时填写 |
| 采购单价 | 否 | 采购口径单价 | 不是销售价格，可为空 |
| 成本单价 | 否 | 成本核算口径单价 | 不是报价模块，可为空 |
| 币种 | 否 | 默认 `CNY` | 后续多币种再扩展 |
| 采购启用 | 否 | 是否可采购 | 后续采购模块会用 |
| 库存启用 | 否 | 是否入库管理 | 后续库存模块会用 |
| 类型属性 | 否 | 选择物料类型后填写 | 字段来自物料属性，不手写 JSON |
| 状态 | 是 | 启用或停用 | 停用后不应再作为新工程配置选择项 |
| 备注 | 否 | 说明用途或维护口径 | 不写临时沟通内容 |

## 常用操作

### 新增物料

1. 点击新增。
2. 选择物料类型。
3. 填写物料编码、中文名称、英文名称。
4. 填写业务类型、主规格、规格摘要、主颜色等基础信息。
5. 选择单位：主单位、采购单位、库存单位、工程使用单位。
6. 如果有供应商和采购信息，填写供应商名称、供应商料号、采购单价、成本单价。
7. 在类型属性区填写当前物料类型对应的参数。
8. 保存。

### 编辑物料

用于补充名称、规格、供应商、价格、单位和类型属性。

物料资料遵守基础资料通用规则：启用状态表示资料已经确认可用于新业务，不允许直接普通修改；停用状态可以普通修改。

点击修改时，系统应先做修改前检查。检查通过才打开编辑抽屉；检查不通过时提示原因，例如“当前资料已启用，请先停用后再编辑”。

如果后续需要修改启用中或已经被引用的物料，必须使用独立的超级修改按钮、独立权限和独立接口，不和普通修改混在一起。

如果物料已经被物料属性、组件明细、工程规则、带出规则或订单快照引用，修改物料编码、物料类型、面料系列和工程使用单位前必须谨慎，必要时先做引用检查。

### 停用物料

停用适用于不再新增使用的物料。

停用后，历史数据仍应保留；后续新增组件包、工程方案和配置规则时不应再选择该物料。

### 删除物料

删除前必须确认没有业务引用。

如果物料已经被物料属性、组件明细、工程方案或配置答案引用，不建议删除，应优先停用。

### 查询物料

推荐优先使用：

- 物料编码。
- 物料名称。
- 物料类型。
- 供应商名称。
- 供应商料号。
- 状态。

采购单价和成本单价第一阶段不作为高频搜索条件。

## 对后续业务的影响

物料资料会影响：

- 面料 SKU：面料本身就是 `FABRIC` 类型物料，必须绑定面料系列。
- 组件包：组件包明细统一引用物料资料。
- 产品工程设计：工程构成项、可选范围、能力规则、带出规则会引用物料。
- 产品配置中心：配置答案后续会间接引用物料或组件包。
- 采购和库存：供应商、供应商料号、采购单位、库存单位、价格字段会被后续 ERP 能力使用。
- 订单快照：订单需要保存物料编码、名称、规格等快照，避免后续资料变动影响历史订单。
- AI 代办：AI 自动录入物料时，需要根据物料类型追问对应参数。

## 常见问题

| 问题 | 原因 | 处理方式 |
| --- | --- | --- |
| 物料类型下拉没有想要的值 | 产品字典没有维护 | 先到产品字典维护 `product_material_type` |
| 类型属性没有出现 | 当前物料类型没有配置物料属性 | 先到物料属性维护适用物料类型和属性字段 |
| 不知道单位怎么选 | 单位管理没有准备好或单位类型不清楚 | 先维护单位管理，再选择采购、库存、工程使用单位 |
| 不知道规格摘要怎么写 | 规格摘要是给人看的可读摘要 | 优先由主规格和类型属性拼出，人工只做微调 |
| 采购单价和成本单价是不是销售价格 | 不是 | 这是物料采购和成本口径，销售价格后续由价格模块处理 |
| 物料资料和物料属性有什么区别 | 物料资料是一条具体物料，类型属性是这条物料的参数值 | 普通录入优先在物料资料抽屉里一起填 |

## 如何确认维护成功

- 物料能在列表中查到。
- 列表能看到物料编码、名称、类型、规格摘要、供应商、供应商料号、采购单价、成本单价。
- 详情或编辑时能看到当前物料类型对应的类型属性值。
- 启用物料能在组件明细、工程设计等选择器里被选到；`FABRIC` 物料可按面料系列过滤。
- 停用物料不再作为新增业务的优先选项。

## AI 代办能力

### 支持的意图

```text
意图名称：新增物料资料
用户说法：帮我新增一个 6米双银色圆管，供应商料号 1116592，采购价 9.57
```

```text
意图名称：补充物料供应商信息
用户说法：把 H0003430 的供应商料号改成 1116592，供应商是某某铝材厂
```

```text
意图名称：停用物料
用户说法：把这个旧款拉珠系统停用，不要再给新工程方案选
```

### 必填槽位

- 物料编码。
- 物料中文名称。
- 物料类型。

### 可推断字段

- 状态默认启用。
- 币种默认 `CNY`。
- 采购启用和库存启用可根据物料类型给默认建议，但保存前必须展示确认。
- 工程使用单位可根据物料类型建议，例如面料可建议米或平方米，配件可建议个，系统包可建议套。

### 需要追问

- 用户没有说明物料类型时，必须追问。
- 当前物料类型要求的关键类型属性缺失时，需要追问，例如面料缺颜色、门幅或厚度。
- 用户要新增面料物料但缺少面料系列时，必须追问。
- 用户提供供应商料号但没有供应商名称时，可以先保存料号，也可以追问供应商。
- 用户要求删除已被引用物料时，需要提示风险并建议停用。

### 保存前确认摘要

```text
将新增物料资料：
- 物料编码：H0003430
- 物料名称：6米/支 双银色圆管
- 物料类型：铝材
- 标准规格：6米/支 双
- 供应商料号：1116592
- 采购单价：9.57 CNY
- 工程使用单位：米
是否确认保存？
```
