## OSS 上传接口

| 动作 | 方法 | 路径 | 说明 |
| --- | --- | --- | --- |
| 上传文件本体 | POST | `/system/oss/upload` | 返回 `ossId`、`url`、`fileName` 等上传结果 |

资料资产只保存业务 metadata，文件本体仍由 `sys_oss` 和 OSS 客户端管理。

## 数据表

| 表名 | 用途 |
| --- | --- |
| `sys_oss` | 文件本体和 OSS 元信息 |

| `pc_media_binding` | 资料资产和业务对象绑定关系 |

## 选择器来源

| 页面字段 | 数据来源 | 过滤条件 |
| --- | --- | --- |
| 资料类型 | 产品字典 `product_asset_type` | 启用字典项 |
| 用途 | 前端固定枚举 | `INSTALL_GUIDE`、`SPEC`、`SWATCH`、`REFERENCE` |
| 可见性 | 前端固定枚举 | `INTERNAL`、`PUBLIC` |

## 状态和字典

| 字段 | 来源 | 可选值 |
| --- | --- | --- |
| `assetType` | 产品字典 `product_asset_type` | 以字典项为准 |
| `usageType` | 前端固定枚举 | `INSTALL_GUIDE`、`SPEC`、`SWATCH`、`REFERENCE` |
| `visibility` | 前端固定枚举 | `INTERNAL`、`PUBLIC` |
| `targetType` | 业务对象类型 | `MATERIAL`、`COMPONENT`、`FABRIC_SERIES` |
| `status` | 基础资料状态规则 | `ENABLED`、`DISABLED` |

## 校验规则

- 资料资产新增需要资产编码和中文名称。
- 资料绑定新增需要资产 ID、资产编码、目标类型、目标编码。
- 资料资产普通修改前调用 `edit-check`。
- 后端资料资产普通修改会校验当前状态；启用状态不可普通编辑。
- 资料绑定当前没有单独 `edit-check` 接口。
- 通用上传文件大小限制为 20MB。

## 常见错误

| 错误场景 | 后端提示 | 用户提示 |
| --- | --- | --- |
| 文件过大 | 前端拦截 | 文件不能超过 20MB。 |
| 资料资产已被绑定 | `product.mediaAsset.hasReferences` | 当前资料已被业务对象引用，建议停用，不建议删除。 |
| 启用状态普通修改 | 当前资料已启用，不可直接修改 | 请先停用后再编辑。 |

## AI 工具调用契约

### 意图

```text
新增资料资产
绑定资料资产
停用资料资产
查询资料绑定
```

### 调用接口和权限

| 意图 | 方法 | 路径 | 权限 |
| --- | --- | --- | --- |
| 新增资料资产 | POST | `/product-capability/media-assets` | `product:asset:upload` |
| 绑定资料资产 | POST | `/product-capability/media-bindings` | `product:asset:bind` |
| 停用资料资产 | PUT | `/product-capability/media-assets/change-status/{id}/{status}` | `product:asset:upload` |
| 查询资料绑定 | GET | `/product-capability/media-bindings/list` | `product:asset:list` |

### 必填槽位

- 新增资产：assetCode、assetNameCn。
- 新增绑定：assetId 或 assetCode、targetType、targetCode。

### 可选槽位

- assetType
- usageType
- languageCode
- visibility
- ossId
- url
- versionNo
- requiredForPublish
- sortOrder

### 可推断字段

- `visibility` 默认 `INTERNAL`。
- `languageCode` 默认当前用户语言。
- `versionNo` 默认 1。
- `status` 默认 `ENABLED`。
- `usageType` 可从“安装说明、规格书、色卡、参考资料”等用户说法推断。

### 需要追问

- 缺少资料资产。
- 缺少目标类型或目标编码。
- 用户要求上传文件但未提供文件。

### Payload 边界

- 只调用已有后端接口，不直接写数据库。
- 不要求用户提供内部审计字段、租户字段或 JSON。
- 不承诺删除 OSS 文件本体，除非后续有明确接口。

### 禁止暴露字段

- assetId
- bindingId
- ossId
- createBy
- createTime
- updateBy
- updateTime

### 保存前确认格式

```text
请确认资料绑定：
- 资产：assetCode
- 目标：targetType / targetCode
- 用途：usageType
- 可见性：visibility
确认后保存。
```

### 失败提示

| 场景 | 给用户的提示 |
| --- | --- |
| 文件上传失败 | 文件上传失败，请检查文件大小或稍后重试。 |
| 资产不存在 | 没找到该资料资产，请先上传或维护资料资产。 |
| 当前资料不可编辑 | 当前资料资产已启用，请先停用后再编辑。 |
