# 基础字典接口与数据契约

## 页面与权限

| 项目 | 值 |
| --- | --- |
| 菜单 | 基础信息 / 基础字典 |
| 路由 | `/product-master/product-dicts` |
| 组件 | `product-center/product-dicts` |
| 权限码 | `product:dict:*` |

## 数据库表

| 表名 | 用途 |
| --- | --- |
| `pc_product_dict_type` | 产品业务字典类型 |
| `pc_product_dict_item` | 产品业务字典项 |

| 字典项 | 分页查询 | `GET /product-capability/product-dict-items/list` |

## 校验规则

- 字典类型编码在同一租户下唯一。
- 同一字典类型下字典项值唯一。
- 启用状态资料不能直接编辑，先停用再编辑。
- 被业务引用的字典类型或字典项不能删除。
- `parent_value` 只表示同一字典类型内部的上级字典项。
