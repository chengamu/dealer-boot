# 产品字典操作手册

## 功能定位

产品字典用于维护产品业务下拉项，例如物料类型、业务类型、组件包类型、资料类型、工程规则类型和配置输入类型。

它不等同于系统字典。系统字典只给后台框架使用，产品配置中心业务枚举统一维护在产品字典里。

## 使用对象

- 技术科：维护工程配置、基础资料录入时需要的业务枚举。
- 开发和测试：确认页面下拉来源、状态和选项是否正确。
- AI 代办能力：后续自动录入物料、面料、工程规则时读取可用枚举。

## 录入前准备

产品字典分两层：

- 字典类型：例如 `product_material_type`、`engineering_rule_type`。
- 字典项：例如 `FABRIC`、`MOTOR`、`THICKNESS_LIMIT`。

具体单位值不在这里维护。门幅单位、厚度单位、采购单位、库存单位、使用单位等都去“单位管理”维护。

“上级字典项”只用于同一个字典类型内部存在父子层级或级联关系的场景。多数产品字典不需要上级字典项，应保持为空并在页面上隐藏。

示例：

| 场景 | 是否需要上级字典项 | 说明 |
| --- | --- | --- |
| `product_material_type` 里的 `FABRIC`、`MOTOR` | 不需要 | 这些是并列物料类型。 |
| `engineering_rule_type` 里的 `DISABLE`、`BLOCK` | 不需要 | 这些是并列规则动作。 |
| 后续如果维护“颜色大类 -> 颜色值” | 需要 | 颜色值可以挂在同一字典类型下的颜色大类下面。 |

上级字典项不是“上级字典类型”，也不是数据库 ID。它应该从当前字典类型已有字典项中选择，展示字典项名称，保存稳定字典值。

## 推荐维护顺序

1. 进入 `基础信息 / 产品字典`。
2. 先确认左侧字典类型是否存在。
3. 选择字典类型后，在右侧维护字典项。
4. 新增字典项时填写业务值、中文标签、英文标签、排序和状态。
5. 停用不再使用的字典项，停用后业务下拉不再返回该项。

## 关键字段说明

| 字段 | 说明 |
| --- | --- |
| 类型编码 | 稳定业务编码，前端和后端按它读取 options。 |
| 类型中文名称 / 英文名称 | 字典类型展示名称。 |
| 业务域 | `BASE`、`ENGINEERING`、`CONFIG`，用于区分基础信息、工程配置、产品配置。 |
| 字典值 | 业务保存值，必须稳定，不能随便改。 |
| 中文标签 / 英文标签 | 下拉展示文案。 |
| 上级字典项 | 可选字段，只在有父子层级的字典类型中使用。普通并列字典项不填写。 |
| 系统内置 | 种子内置项，不能删除。 |
| 可编辑 | 控制是否允许维护基础信息。 |
| 状态 | `ENABLED` 返回到下拉，`DISABLED` 不返回到下拉。 |

## 常见错误

- 不要把 `MM`、`M`、`PCS` 等具体单位放进产品字典，具体单位只在 `pc_unit` 维护。
- 不要把产品业务枚举写到 `sys_dict_type` / `sys_dict_data`。
- 不要删除系统内置字典项；如业务不再使用，优先停用。
- 修改字典值前先确认已有业务数据是否保存了该值。
- 普通字典项不要填写上级字典项；只有明确需要父子层级或级联选择时才使用。

## AI 代办能力

### 支持的意图

```text
意图名称：新增产品业务字典项
用户说法：帮我在物料类型里新增一个太阳能板
```

```text
意图名称：停用产品业务字典项
用户说法：把这个旧的工程规则类型停用
```

```text
意图名称：查询产品字典项
用户说法：现在物料类型有哪些选项
```

### 调用入口

- 页面：基础信息 / 产品字典
- 接口：`POST /product-capability/product-dict-items`
- 权限：`product:dict:add`

### 必填槽位

- 字典类型编码
- 字典值
- 中文标签

### 可推断字段

- 状态默认 `ENABLED`
- 系统内置默认 `false`
- 可编辑默认 `true`
- 业务域可从字典类型推断

### 需要追问

- 找不到字典类型编码时必须追问或先创建字典类型。
- 缺少英文标签时可以追问，也可以先用中文标签临时填充并提示后续补齐。
- 用户要求新增具体单位时，需要提示应去单位管理维护。

### 失败提示

| 场景 | 给用户的提示 |
| --- | --- |
| 字典类型不存在 | 没有找到对应字典类型，请先确认类型编码或先创建字典类型。 |
| 字典项值重复 | 该字典项值已存在，请换一个值，或者编辑已有字典项。 |
| 具体单位误放字典 | 具体单位请到单位管理维护，不放在产品字典里。 |

### 保存前确认摘要

```text
将新增产品字典项：
- 字典类型：product_material_type / 物料类型
- 字典值：SOLAR_PANEL
- 中文标签：太阳能板
- 英文标签：Solar Panel
- 状态：启用
是否确认保存？
```
