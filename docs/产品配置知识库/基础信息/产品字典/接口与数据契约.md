# 产品字典接口与数据契约

## 页面与权限

| 项目 | 值 |
| --- | --- |
| 菜单 | 基础信息 / 产品字典 |
| 路由 | `/product-master/product-dicts` |
| 组件 | `product-center/product-dicts` |
| 查询权限 | `product:dict:list` |
| 新增权限 | `product:dict:add` |
| 编辑权限 | `product:dict:edit` |
| 删除权限 | `product:dict:remove` |
| 引用检查权限 | `product:dict:reference` |

## 数据库表

### `pc_product_dict_type`

| 字段 | 说明 |
| --- | --- |
| `dict_type_id` | 字典类型 ID |
| `tenant_id` | 租户 ID |
| `dict_type_code` | 字典类型编码 |
| `dict_type_name_cn` | 类型中文名称 |
| `dict_type_name_en` | 类型英文名称 |
| `business_domain` | 业务域：`BASE` / `ENGINEERING` / `CONFIG` |
| `system_flag` | 系统内置 |
| `editable_flag` | 可编辑 |
| `status` | `ENABLED` / `DISABLED` |
| `sort_order` | 排序 |
| `del_flag` | 逻辑删除标记 |

唯一约束：

```text
tenant_id + dict_type_code + del_flag='0'
```

### `pc_product_dict_item`

| 字段 | 说明 |
| --- | --- |
| `dict_item_id` | 字典项 ID |
| `tenant_id` | 租户 ID |
| `dict_type_code` | 字典类型编码 |
| `dict_item_value` | 字典项业务值 |
| `dict_item_label_cn` | 中文标签 |
| `dict_item_label_en` | 英文标签 |
| `parent_value` | 上级字典值，预留级联 |
| `system_flag` | 系统内置 |
| `editable_flag` | 可编辑 |
| `status` | `ENABLED` / `DISABLED` |
| `sort_order` | 排序 |
| `del_flag` | 逻辑删除标记 |

唯一约束：

```text
tenant_id + dict_type_code + dict_item_value + del_flag='0'
```

## 主要接口

### 字典类型

| 动作 | 方法和路径 |
| --- | --- |
| 分页查询 | `GET /product-capability/product-dict-types/list` |
| options | `GET /product-capability/product-dict-types/options` |
| 详情 | `GET /product-capability/product-dict-types/{id}` |
| 新增 | `POST /product-capability/product-dict-types` |
| 修改 | `PUT /product-capability/product-dict-types` |
| 启停 | `PUT /product-capability/product-dict-types/change-status/{id}/{status}` |
| 删除 | `DELETE /product-capability/product-dict-types/{ids}` |
| 引用检查 | `GET /product-capability/product-dict-types/{id}/references` |

### 字典项

| 动作 | 方法和路径 |
| --- | --- |
| 分页查询 | `GET /product-capability/product-dict-items/list` |
| options | `GET /product-capability/product-dict-items/options` |
| 按类型 options | `GET /product-capability/product-dict-items/type/{dictTypeCode}` |
| 详情 | `GET /product-capability/product-dict-items/{id}` |
| 新增 | `POST /product-capability/product-dict-items` |
| 修改 | `PUT /product-capability/product-dict-items` |
| 启停 | `PUT /product-capability/product-dict-items/change-status/{id}/{status}` |
| 删除 | `DELETE /product-capability/product-dict-items/{ids}` |
| 引用检查 | `GET /product-capability/product-dict-items/{id}/references` |

## Options 响应

`GET /product-capability/product-dict-items/type/{dictTypeCode}` 只返回启用字典项。

```json
[
  {
    "dictTypeCode": "product_material_type",
    "value": "FABRIC",
    "label": "面料",
    "labelCn": "面料",
    "labelEn": "Fabric",
    "parentValue": null
  }
]
```

## 下拉来源规则

| 场景 | 来源 |
| --- | --- |
| 物料类型 | `product_material_type` |
| 业务类型 | `product_business_type` |
| 组件包类型 | `product_component_type` |
| 资料类型 | `product_asset_type` |
| 工程构成项类型 | `engineering_item_type` |
| 工程可选范围类型 | `engineering_scope_type` |
| 工程规则来源 | `engineering_rule_source` |
| 工程规则类型 | `engineering_rule_type` |
| 工程规则操作符 | `engineering_rule_operator` |
| 工程规则动作 | `engineering_rule_action` |
| 工程带出类型 | `engineering_output_type` |
| 工程数量模式 | `engineering_qty_mode` |
| 工程严重级别 | `engineering_severity` |
| 配置输入类型 | `config_input_type` |
| 配置答案来源 | `config_option_source_type` |
| 具体单位值 | `pc_unit`，不走产品字典 |

## 校验和错误

| 场景 | 错误 key |
| --- | --- |
| 类型编码为空 | `product.dict.typeCodeRequired` |
| 类型编码重复 | `product.dict.typeCodeExists` |
| 字典类型不存在 | `product.dict.typeNotFound` |
| 字典项值为空 | `product.dict.itemValueRequired` |
| 字典项值重复 | `product.dict.itemValueExists` |
| 删除系统内置类型 | `product.dict.systemTypeCannotDelete` |
| 删除有字典项的类型 | `product.dict.typeHasItems` |
| 删除系统内置字典项 | `product.dict.systemItemCannotDelete` |
