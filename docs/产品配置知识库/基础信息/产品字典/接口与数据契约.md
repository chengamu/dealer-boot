## Options 响应

`GET /product-capability/product-dict-items/type/{dictTypeCode}` 只返回启用字典项。

```json
[
  {
    "dictTypeCode": "product_material_type",
    "value": "FABRIC",
    "label": "面料",
    "labelCn": "面料",
    "labelEn": "Fabric",
    "parentValue": null
  }
]
```

## 上级字典项规则

`parent_value` 的语义是“当前字典项挂在哪个同类型字典项下面”，不是上级字典类型，也不是数据库 ID。

页面和接口应遵守：

- 默认不展示 `parent_value` 列和输入项，除非该字典类型明确支持层级。
- 展示时字段名称使用“上级字典项”，不要使用“上级字典值”。
- 录入时使用选择器，只能选择当前 `dict_type_code` 下已启用的字典项。
- 列表展示父级字典项中文标签，必要时 tooltip 展示保存值。
- 普通并列字典项保存 `null` 或空字符串。
- 删除或停用父级字典项前，需要检查是否存在子字典项引用。

## 校验和错误

| 场景 | 错误 key |
| --- | --- |
| 类型编码为空 | `product.dict.typeCodeRequired` |
| 类型编码重复 | `product.dict.typeCodeExists` |
| 字典类型不存在 | `product.dict.typeNotFound` |
| 字典项值为空 | `product.dict.itemValueRequired` |
| 字典项值重复 | `product.dict.itemValueExists` |
| 删除系统内置类型 | `product.dict.systemTypeCannotDelete` |
| 删除有字典项的类型 | `product.dict.typeHasItems` |
| 删除系统内置字典项 | `product.dict.systemItemCannotDelete` |

### 必填槽位

新增字典类型：

- `dictTypeCode`
- `dictTypeNameCn`
- `businessDomain`

新增字典项：

- `dictTypeCode`
- `dictItemValue`
- `dictItemLabelCn`

### 可选槽位

- `dictTypeNameEn`
- `dictItemLabelEn`
- `parentValue`
- `sortOrder`
- `remark`

### 可推断字段

- `status = ENABLED`
- `systemFlag = false`
- `editableFlag = true`
- `businessDomain` 可从字典类型用途推断，但保存前必须展示确认。
- `dictItemLabelEn` 可暂用中文标签占位，后续人工校对。

### 需要追问

- 用户只说新增选项，但没有说明字典类型时，需要追问。
- 字典类型不存在时，需要追问是先创建字典类型，还是选择已有字典类型。
- 用户新增 `MM`、`M`、`PCS` 等具体单位时，需要提示改去单位管理维护。
- 用户要求删除系统内置项时，需要提示不可删除，可讨论是否停用。

### Payload 边界

- 不要求用户提供 `dictTypeId`、`dictItemId`、租户字段或审计字段。
- `parentValue` 只用于同一字典类型内的父子层级，不表示上级字典类型。
- 字典值必须是稳定业务编码，不能使用会频繁变化的展示文案。

### 禁止暴露字段

- `dictTypeId`
- `dictItemId`
- `tenantId`
- `delFlag`
- `createBy`
- `createTime`
- `updateBy`
- `updateTime`

### 保存前确认格式

```text
将新增产品字典项：
- 字典类型：{dictTypeCode} / {dictTypeNameCn}
- 字典值：{dictItemValue}
- 中文标签：{dictItemLabelCn}
- 英文标签：{dictItemLabelEn}
- 上级字典项：{parentValueLabel}
- 状态：启用
是否确认保存？
```

### 失败提示

| 场景 | 给用户的提示 |
| --- | --- |
| 字典类型不存在 | 没有找到对应字典类型，请先确认类型编码，或先创建字典类型。 |
| 字典项值重复 | 该字典项值已存在，请换一个值，或者编辑已有字典项。 |
| 系统内置项不可删除 | 这是系统内置字典项，不能删除；如果业务不用，建议停用。 |
| 字典类型已有字典项 | 当前字典类型下已有字典项，不建议删除字典类型。 |
| 具体单位误放字典 | 具体单位请到单位管理维护，不放在产品字典里。 |
| 无权限 | 当前账号没有维护产品字典的权限，请联系管理员授权。 |
