# 物料类型操作手册

物料类型用于维护两级基础主数据：属性分组和分组下的物料类型。属性分组决定一类物料共享哪些属性模板，物料类型用于物料管理录入时选择具体类型。

## 推荐维护顺序

1. 维护属性分组，例如面料、铝材、控制系统、五金配件、包装参数。
2. 在分组下维护物料类型，例如控制系统下维护电机、遥控器、拉珠。
3. 到物料属性维护该分组共享的属性模板。
4. 到物料管理新增物料并选择物料类型。

## 关键规则

- 物料类型必须归属一个属性分组。
- 同一分组下的物料类型共享该分组的物料属性模板。
- 系统内置分组和类型用于基础业务口径，不建议删除。
- 已被物料或其他业务引用的分组、类型不能删除，优先停用。

## 人工测试重点

- 新增属性分组后，右侧可以维护该分组下的物料类型。
- 新增物料类型后，物料管理新增页面可以选择该类型。
- 选择物料类型后，物料管理自动带出属性分组，并加载该分组的物料属性。
- 停用物料类型后，新增物料下拉不再出现该类型。
