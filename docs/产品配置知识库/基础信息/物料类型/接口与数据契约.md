## 数据表

| 表名 | 用途 |
| --- | --- |
| `pc_material_type_group` | 属性分组主数据 |
| `pc_material_type` | 物料类型主数据 |

## 关键字段

| 表 | 字段 | 说明 |
| --- | --- | --- |
| `pc_material_type_group` | `group_code` | 属性分组编码 |
| `pc_material_type_group` | `group_name_cn` / `group_name_en` | 属性分组名称 |
| `pc_material_type` | `material_type_code` | 物料类型编码 |
| `pc_material_type` | `material_type_name_cn` / `material_type_name_en` | 物料类型名称 |
| `pc_material_type` | `attribute_group_id` / `attribute_group_code` / `attribute_group_name_cn` | 所属属性分组快照 |
| 两表 | `system_flag` / `editable_flag` | 系统内置和可编辑控制 |
| 两表 | `status` | `ENABLED` / `DISABLED` |

## 校验规则

- 属性分组编码在同一租户下唯一。
- 物料类型编码在同一租户下唯一。
- 物料类型必须归属启用属性分组。
- 启用状态资料不能直接编辑，先停用再编辑。
- 被物料类型、物料属性或物料引用的属性分组不能删除。
- 被物料引用的物料类型不能删除。

### 必填槽位

新增属性分组：

- `groupCode`：属性分组编码。
- `groupNameCn`：属性分组中文名称。

新增物料类型：

- `materialTypeCode`：物料类型编码。
- `materialTypeNameCn`：物料类型中文名称。
- `attributeGroupCode` 或可唯一识别的属性分组名称。

### 可选槽位

- `groupNameEn`：属性分组英文名称。
- `materialTypeNameEn`：物料类型英文名称。
- `sortOrder`：排序。
- `remark`：备注。

### 可推断字段

- 新增属性分组和物料类型默认 `status = ENABLED`。
- 物料类型的属性分组快照字段由后端根据属性分组回填。

### 需要追问

- 物料类型缺少所属属性分组。
- 用户说的属性分组无法唯一匹配。
- 用户要求编辑启用状态资料，但没有先停用。
- 用户要求删除已引用分组或物料类型。

### Payload 边界

- AI 不提交 `tenantId`、`createBy`、`createTime`、`updateBy`、`updateTime`、`delFlag`。
- AI 不让用户填写内部 ID。
- 编辑前必须先调用对应 `edit-check`。
- AI 不在物料类型模块维护具体物料属性字段；具体字段应进入物料属性模块。

### 禁止暴露字段

- `groupId`
- `materialTypeId`
- `tenantId`
- `createBy`
- `createTime`
- `updateBy`
- `updateTime`
- `delFlag`

### 保存前确认格式

```text
将新增物料类型：
- 所属属性分组：控制系统
- 物料类型编码：REMOTE
- 物料类型名称：遥控器
- 状态：启用
是否确认保存？
```

### 失败提示

| 场景 | 用户提示 |
| --- | --- |
| 属性分组缺失 | 请说明物料类型属于哪个属性分组。 |
| 属性分组不存在 | 没找到这个属性分组，请先新增属性分组。 |
| 分组编码重复 | 属性分组编码已存在，请换一个编码或编辑已有分组。 |
| 物料类型编码重复 | 物料类型编码已存在，请换一个编码或编辑已有类型。 |
| 启用状态不可普通编辑 | 该资料已启用，请先停用后再编辑。 |
| 删除已引用资料 | 该资料已被引用，不能删除；可以停用让它不再进入新增选择。 |
