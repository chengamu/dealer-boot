# 面料系列接口与数据契约

## 页面与权限

| 项目 | 内容 |
| --- | --- |
| 菜单名称 | 基础信息 / 面料系列 |
| 前端路由 | `/product-master/fabrics`、`/product-master/fabric-series` |
| 前端组件 | `admin-ui/src/pages/product-center/fabric/ProductFabricPage.vue` |
| 前端 API | `admin-ui/src/api/product-capability/fabric.ts` |
| Controller | `ProductFabricController` |
| Service | `FabricSeriesService` |
| 权限码 | `product:fabric:list`、`product:fabric:add`、`product:fabric:edit`、`product:fabric:remove`、`product:fabric:reference` |

## 后端接口

| 动作 | 方法 | 路径 | 权限 |
| --- | --- | --- | --- |
| 列表 | GET | `/product-capability/fabric-series/list` | `product:fabric:list` |
| 选项 | GET | `/product-capability/fabric-series/options` | `product:fabric:list` |
| 详情 | GET | `/product-capability/fabric-series/{id}` | `product:fabric:list` |
| 修改前检查 | GET | `/product-capability/fabric-series/{id}/edit-check` | `product:fabric:edit` |
| 新增 | POST | `/product-capability/fabric-series` | `product:fabric:add` |
| 修改 | PUT | `/product-capability/fabric-series` | `product:fabric:edit` |
| 启停 | PUT | `/product-capability/fabric-series/change-status/{id}/{status}` | `product:fabric:edit` |
| 删除 | DELETE | `/product-capability/fabric-series/{ids}` | `product:fabric:remove` |
| 引用检查 | GET | `/product-capability/fabric-series/{id}/references` | `product:fabric:reference` |

## 数据表

| 表名 | 用途 |
| --- | --- |
| `pc_fabric_series` | 面料系列主数据和系列级限制 |
| `pc_material` | 统一物料主表；`material_type = FABRIC` 的记录是面料 SKU，并保存系列快照字段 |
| `pc_material_attribute` | 面料 SKU 的颜色、门幅、厚度、克重等属性值 |
| `pc_media_binding` | 系列色卡、系列资料绑定 |

## 关键字段

### `pc_fabric_series`

| 字段 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| `series_id` | bigint | 系统生成 | 系列内部 ID |
| `series_code` | varchar | 是 | 系列编码 |
| `series_name_cn` | varchar | 是 | 系列中文名称 |
| `series_name_en` | varchar | 否 | 系列英文名称 |
| `material_type` | varchar | 否 | 当前默认 `FABRIC` |
| `default_thickness_unit` | varchar | 否 | 默认厚度单位 |
| `thickness_rule_enabled` | boolean | 否 | 厚度规则开关 |
| `max_thickness_diff` | numeric | 否 | 最大厚度差 |
| `max_combined_thickness` | numeric | 否 | 最大组合厚度 |
| `width_rule_enabled` | boolean | 否 | 门幅规则开关 |
| `available_widths` | varchar | 否 | 可选门幅文本 |
| `min_width_value` | numeric | 否 | 最小门幅 |
| `max_width_value` | numeric | 否 | 最大门幅 |
| `width_unit` | varchar | 否 | 门幅单位 |
| `status` | varchar | 是 | `ENABLED` / `DISABLED` |

### `pc_material` 面料系列快照

| 字段 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| `fabric_series_id` | bigint | `FABRIC` 必填 | 面料系列 ID |
| `fabric_series_code` | varchar | `FABRIC` 必填 | 面料系列编码快照 |
| `fabric_series_name_cn` | varchar | 否 | 面料系列中文名称快照 |

## 选择器来源

| 页面字段 | 数据来源 | 过滤条件 |
| --- | --- | --- |
| 物料资料 / 面料系列 | `GET /product-capability/fabric-series/options` | `status=ENABLED`、`materialType=FABRIC` |
| 面料物料列表 | `GET /product-capability/materials/list` | `materialType=FABRIC`，可叠加 `fabricSeriesCode` |

## 校验规则

- 面料系列新增和修改必须提供系列编码、中文名称。
- `FABRIC` 物料必须选择面料系列。
- 非 `FABRIC` 物料保存时后端会清空面料系列快照字段。
- 面料系列被面料物料或系列资料绑定引用时，删除前引用检查不通过。
- 唯一性优先在接口服务层校验，不把“名称 + 规格 + 供应商”等业务规则写死为数据库唯一约束。
