# 面料系列操作手册

## 功能定位

面料系列用于维护面料物料录入前必须准备的系列主数据。

具体面料 SKU 不在本页面维护；面料 SKU 统一在物料管理中维护，物料类型选择 `FABRIC`，并绑定一个面料系列。

系列页面可以上传系列说明书、整本目录、系列介绍等系列级参考资料。具体颜色色卡、检测报告、实物图片等资料应绑定到具体面料物料。

```text
基础字典 / 单位管理 / 物料属性 -> 面料系列 -> 物料管理（FABRIC） -> 产品工程设计 -> 产品配置中心
```

## 使用前准备

- 基础字典：物料类型中存在 `FABRIC`。
- 物料属性：为 `FABRIC` 配置颜色、门幅、厚度、克重、材质等属性。

## 页面入口

- 菜单：基础信息 / 面料系列。
- 路由：`/product-master/fabric-series`。

## 字段说明

| 字段 | 是否必填 | 说明 |
| --- | --- | --- |
| 系列编码 | 是 | 面料系列稳定业务编码。 |
| 系列中文名称 | 是 | 业务人员识别用名称。 |
| 系列英文名称 | 否 | 英文环境展示。 |
| 状态 | 是 | `ENABLED` / `DISABLED`。 |

## 常用操作

### 新增面料系列

1. 打开面料系列页面。
2. 填写系列编码和中文名称。
3. 保存并启用。

### 新增面料物料

1. 打开物料管理。
2. 物料类型选择 `FABRIC`。
3. 选择面料系列。
4. 填写物料编码、名称、规格、供应商、样册编号和类型属性。
5. 保存。

### 上传系列参考资料

在面料系列详情中上传系列说明书、整本目录或系列介绍。上传后资料绑定目标类型为 `FABRIC_SERIES`，用途默认为参考资料。

具体色卡应在物料管理中选择具体面料物料后上传，绑定目标类型为 `MATERIAL`。

## 删除或停用

面料系列已被面料物料或系列级参考资料引用时，不建议删除，应优先停用。

## 如何确认维护成功

- 面料系列能在列表中查到。
- 新增 `FABRIC` 物料时能选择该系列。
- 查询面料物料时可按 `materialType=FABRIC` 和面料系列过滤。

## AI 代办能力

### 支持的意图

```text
新增面料系列
编辑面料系列
停用面料系列
查询面料系列
```

### 必填槽位

- 系列编码。
- 系列中文名称。

### 需要追问

- 缺少系列编码。
- 缺少系列名称。
- 用户要新增面料 SKU 但没有面料系列时，应先引导创建系列或选择已有系列。

### 保存前确认摘要

```text
将新增面料系列：
- 系列编码：seriesCode
- 系列名称：seriesNameCn
确认后保存。
```
