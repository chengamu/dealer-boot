# 面料系列操作手册

## 功能定位

面料系列用于维护面料物料录入前必须准备的系列主数据，以及系列级门幅、厚度等原始限制。

具体面料 SKU 不在本页面维护；面料 SKU 统一在物料资料中维护，物料类型选择 `FABRIC`，并绑定一个面料系列。

```text
产品字典 / 单位管理 / 物料属性定义 -> 面料系列 -> 物料资料（FABRIC） -> 产品工程设计 -> 产品配置中心
```

## 使用前准备

- 产品字典：物料类型中存在 `FABRIC`。
- 单位管理：维护门幅、厚度、工程使用等单位。
- 物料属性定义：为 `FABRIC` 配置颜色、门幅、厚度、克重、材质等属性。

## 页面入口

- 菜单：基础信息 / 面料系列。
- 路由：`/product-master/fabric-series`，也可从 `/product-master/fabrics` 进入。

## 字段说明

| 字段 | 是否必填 | 说明 |
| --- | --- | --- |
| 系列编码 | 是 | 面料系列稳定业务编码。 |
| 系列中文名称 | 是 | 业务人员识别用名称。 |
| 系列英文名称 | 否 | 英文环境展示。 |
| 默认厚度单位 | 否 | 来自单位管理。 |
| 厚度规则开关 | 否 | 记录系列原始厚度限制，不替代工程规则。 |
| 最大厚度差 | 否 | 与厚度规则配套使用。 |
| 最大组合厚度 | 否 | 与厚度规则配套使用。 |
| 门幅规则开关 | 否 | 记录系列原始门幅限制。 |
| 可选门幅 | 否 | 当前保存文本，建议统一格式。 |
| 最小门幅 / 最大门幅 | 否 | 与门幅单位一起理解。 |
| 门幅单位 | 否 | 来自单位管理。 |
| 状态 | 是 | `ENABLED` / `DISABLED`。 |

## 常用操作

### 新增面料系列

1. 打开面料系列页面。
2. 填写系列编码和中文名称。
3. 按业务需要维护门幅、厚度限制。
4. 保存并启用。

### 管理面料物料

在面料系列列表点击“管理面料物料”，系统跳转到物料资料，并带入 `materialType=FABRIC` 和当前 `fabricSeriesCode` 过滤条件。

新增面料物料时：

1. 打开物料资料。
2. 物料类型选择 `FABRIC`。
3. 选择面料系列。
4. 填写物料编码、名称、规格、供应商、样册编号和类型属性。
5. 保存。

## 删除或停用

面料系列已被面料物料或系列资料绑定引用时，不建议删除，应优先停用。

## 如何确认维护成功

- 面料系列能在列表中查到。
- 新增 `FABRIC` 物料时能选择该系列。
- 从系列行进入“管理面料物料”后，物料列表按当前系列过滤。

## AI 代办能力

### 支持的意图

```text
新增面料系列
编辑面料系列
停用面料系列
查询面料系列
```

### 必填槽位

- 系列编码。
- 系列中文名称。

### 需要追问

- 缺少系列编码。
- 缺少系列名称。
- 用户要新增面料 SKU 但没有面料系列时，应先引导创建系列或选择已有系列。

### 保存前确认摘要

```text
将新增面料系列：
- 系列编码：seriesCode
- 系列名称：seriesNameCn
- 门幅规则：widthRuleEnabled
- 厚度规则：thicknessRuleEnabled
确认后保存。
```
