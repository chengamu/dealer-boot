# 配方管理操作手册

## 录入顺序

1. 在基础信息完成产品分类、产品字典、单位、厂家、物料类型、物料属性和物料管理。
2. 在配方档案新增配方，填写编号、名称、产品分类、产品类型和尺寸范围。
3. 在配方原料选入生产可能用到的物料，确认生产必带、生产备注和用量规则。
4. 在业务配置项维护客户下单时能选择的配置项、选项值、关联生产物料和异常限制。
5. 在配方模拟输入样例宽高和选项，确认 BOM 与用量结果。
6. 校验通过后提交审核。
7. 审核通过后生成版本快照，供后续报价、订单和生产引用。

## 配方档案

配方档案只维护配方主信息，不维护物料明细。

必填信息：

- 配方编号
- 配方名称
- 产品分类
- 产品类型
- 尺寸范围

产品类型来自产品字典，不在配方里手工造值。

## 配方原料

配方原料是技术科确认生产清单的核心入口。

### 物料来源

- 只能从物料管理选入已维护物料。
- 不在配方原料里新增基础物料。
- 物料类型、属性分组、规格型号、主单位等来自基础信息。

### 原料字段

| 字段 | 用法 |
| --- | --- |
| 属性分组 | 物料所属大类，例如面料、铝材、系统、配件 |
| 物料类型 | 物料小类，例如遮光面料、电机、下杆 |
| 物料编码/名称/规格型号 | 用于技术科确认是否选对物料 |
| 单位 | 展示物料主单位 |
| 生产必带 | 命中配置时生产必须包含；这是生产属性，不代表客户可见 |
| 用量汇总 | 展示固定值或公式规则摘要 |
| 生产备注 | 给生产端看的说明 |

### 用量设置

用量规则从原料行的“用量设置”进入。物料池 grid 只做材料确认，不在主表格里展开复杂公式。

当前用量方式：

| 用量方式 | 用法 |
| --- | --- |
| 固定值 | 始终固定数量的物料，只维护一个固定用量 |
| 公式规则 | 按默认规则或表达式条件维护多条公式规则 |

固定值示例：

- 安装码固定 `2.00` 个。
- 包装袋固定 `1.00` 个。

公式规则示例：

- 默认规则：公式为 `1`
- 表达式条件：`订单宽 > 2500` 时公式为 `2`
- 表达式条件：`系统 = 电机` 时使用电机相关物料公式

当前公式规则不支持把 `if/case` 写进一个公式里。分支优先放到多条规则的“使用条件”中，每个公式只表达一条计算方式。

### 条件和公式

公式编辑器支持中文变量录入，并转换成内部变量保存。当前代码层已支持的订单变量包括：

| 中文变量 | 内部变量 | 当前使用说明 |
| --- | --- | --- |
| 订单宽 | `orderWidth` | 模拟页输入字段 |
| 订单高 | `orderHeight` | 模拟页输入字段 |
| 订单长 | `orderLength` | 当前模拟计算中按订单宽带入 |
| 订单面积 | `orderArea` | 当前按订单宽乘订单高生成 |
| 订单重量 | `orderWeight` | 当前模拟中默认 `0`，使用前需业务确认 |

条件表达式还支持两类业务条件：

- 业务配置项选择值：系统会把选中的配置项值放入 `option_配置项编码` 变量中，例如可表达“系统 = 电机”。
- 原料池条件：来源于当前配方原料池，不从全系统物料随意取值；当前界面支持按属性分组下的物料类型构造条件，例如“面料.物料类型 = 遮光面料”或“系统.物料类型 = 电机”。

原料池条件用于表达“当前配方原料池中某一类物料怎么处理”。若要按面料系列做扣减，需要先确认面料系列是否已进入当前原料池物料快照或物料属性。

## 业务配置项

业务配置项决定客户下单时能选什么，以及选中后生产端自动带出什么。

示例结构：

```text
系统
├─ 拉珠
│  └─ 拉珠型号
├─ 电机
│  ├─ 电机型号
│  └─ 遥控型号
└─ 无拉
```

配置项说明：

- 根配置项默认展示，例如系统。
- 子配置项通过父选项值控制展示，例如选择系统 = 电机后才展示电机型号和遥控型号。
- 选项值可以关联生产物料，例如选择遥控型号后带出对应遥控器或网关。
- 生产必带件可以由选项值自动带出，但不一定给客户展示。

异常限制只用于禁用或提醒，例如订单宽度过大时禁用拉珠。正常分支展示不要放到异常限制里。

## 配方模拟

配方模拟用于审核前检查。

当前模拟输入包含：

- 订单宽
- 订单高
- 客户选择的配置项值

模拟结果应能看出：

- 哪些物料被带出
- 每个物料的用量结果
- 是否命中异常限制
- 是否存在缺默认规则、缺公式、公式无效等问题

## 配方审核

提交审核前必须先校验通过。

审核通过后：

- 配方状态进入生效。
- 系统生成版本号和版本快照。
- 草稿后续修改不影响已审核版本。

驳回后：

- 配方回到可编辑状态。
- 技术科按驳回原因修改后重新校验和提交。

停用后：

- 配方不再重新启用。
- 继续使用时复制或新建草稿重新审核。
