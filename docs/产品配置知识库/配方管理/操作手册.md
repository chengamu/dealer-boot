# 配方管理操作手册

配方管理用于维护产品配方档案，并在审核通过时冻结一个可追溯版本。当前只有两个入口：配方列表管理和设置配方。

## 使用前准备

先确认以下基础资料已经启用：

- 产品分类。
- 产品字典中的 `product_type` 字典项，例如成品帘、定制帘。
- 物料属性定义中的属性分组和物料类型。
- 物料管理中的可用物料。
- 单位管理中的可用单位。

产品类型来自产品字典，物料分组和物料类型来自基础信息，单位来自单位管理，物料来自物料管理。配方页面不重复维护这些基础资料。

## 推荐录入顺序

1. 新增配方主表，填写配方编号、配方名称、产品分类、产品类型、最大宽度、最大高度。
2. 保存草稿。
3. 选择草稿配方，点击“设置配方”，进入独立设置页面。
4. 在“设置内容”中维护原料池，把本配方能使用的面料、铝材、系统、配件、零件包、包装选入。
5. 在原料池行上维护用量设置。简单物料可用单行用量兜底；像 38 管、罩壳、下杆这类会按不同面料扣减的物料，要维护条件用量规则。
6. 在“业务配置项”中维护下单可选项，例如面料、控制方式、铝材颜色、频道、下杆包布、遮光槽、拉珠颜色、静音。
7. 维护选项值对应的原料池物料，让配置项闭环到具体物料。
8. 维护限制条件，例如订单宽度超过指定值时禁用拉珠或某类面料。
9. 点击“校验配方”，校验通过后提交审核。
10. 审核通过后配方进入生效状态，并自动生成版本号，例如 `V1`。
11. 生效配方不用时执行停用。停用后不重新启用，如需继续使用，重新创建草稿版本审核。

## 字段说明

| 字段 | 是否必填 | 说明 |
| --- | --- | --- |
| 配方编号 | 是 | 稳定业务编号，程序校验唯一。 |
| 配方名称 | 是 | 建议描述产品类别、尺寸边界和关键型号。 |
| 产品分类 | 是 | 来自已启用产品分类。 |
| 产品类型 | 是 | 来自产品字典 `product_type`。 |
| 最大宽度 | 是 | 第一版固定按英寸保存和展示。 |
| 最大高度 | 是 | 第一版固定按英寸保存和展示。 |
| 物料行数 | 否 | 保存设置后由系统维护。 |
| 校验状态 | 否 | 保存设置后变为未校验，执行校验后更新。 |
| 当前版本 | 否 | 审核通过后自动生成，例如 `V1`、`V2`。 |
| 状态 | 否 | 由系统动作流转，不在普通编辑表单中手工选择。 |

## 设置内容

“设置内容”用于维护技术科认可的原料池。它不是订单下单页，而是定义这个配方可使用哪些基础物料。

原料池每行至少需要：

- 物料。
- 单位。
- 有效用量规则。

用量设置从原料池行进入，不做独立菜单。当前支持两层：

- 单行兜底用量：适合纸箱、说明书等简单物料。
- 条件用量规则：适合 38 管、罩壳、下杆、面料等需要按不同面料或配置项值设置扣减和公式的物料。

条件用量规则可以按 `FABRIC` 面料配置项批量生成，一条面料值对应一条规则。每条规则可维护宽度扣减、高度扣减、计量口径、固定用量、公式、计算单位、取整方式、最小用量、最大用量、损耗率和生产备注。存在条件规则的物料必须保留一条默认规则，避免订单选择未命中任何条件时没有兜底。

公式第一版支持变量：`orderWidthMm`、`orderHeightMm`、`widthDeductMm`、`heightDeductMm`、`lossRate`、`fixedUsageQty`。例如 38 管按面料扣减可填写：

```text
(orderWidthMm - widthDeductMm) / 1000
```

## 业务配置项

“业务配置项”用于维护下单时业务能选择什么，以及这些选择会带出哪些原料池物料。

当前包含：

- 配置项定义：例如面料、控制方式、铝材颜色、频道。
- 选项值：例如拉珠、无拉、电机。
- 选项物料：例如选择电机时带出电机、遥控器、对应控制器。
- 订单端显示条件：例如“电机型号”“遥控型号”设置为“系统 = 电机 时显示”，客户未选择电机时不展示。
- 限制条件：例如订单宽度超过阈值时禁用拉珠。

选项物料必须来自当前配方原料池，不能直接选择原料池之外的物料。

配置项顺序就是后续下单选择顺序，可通过上移、下移调整。显示条件只负责“这个选择项什么时候出现”，限制条件只负责“出现后某些值是否禁用或提醒”，两者不要混用。

## 状态规则

| 状态 | 可做动作 |
| --- | --- |
| 草稿 | 可编辑、可删除、可设置配方、可校验、校验通过后可提交审核。 |
| 待审核 | 不可普通编辑、不可删除、不可设置配方，可审核通过或驳回。 |
| 驳回 | 可编辑、可设置配方，重新校验后再提交审核。 |
| 生效 | 不可普通编辑、不可删除，可停用。 |
| 停用 | 保留历史，不可编辑、不可删除、不可重新启用。 |

审核通过会写入审核人、审核时间和当前版本。版本快照记录审核通过时的主表、原料池、条件用量规则、业务配置项、选项物料和限制条件；变更记录记录用户做过哪些动作。

## 常见问题

| 场景 | 处理方式 |
| --- | --- |
| 找不到产品类型 | 到基础字典中维护并启用 `product_type` 字典项。 |
| 找不到物料分组或物料类型 | 到物料属性定义中维护并启用。 |
| 找不到物料或单位 | 到基础信息中维护并启用物料或单位。 |
| 不能提交审核 | 先保存有效原料池和业务配置项，并校验通过。 |
| 待审核不能编辑 | 先驳回后再编辑。 |
| 生效配方不能删除 | 停用保留历史，不删除。 |
| 停用后想继续用 | 新建或复制为草稿，重新设置、校验和审核。 |

## 人工测试重点

- 新增草稿配方。
- 产品类型下拉来自产品字典。
- 点击“设置配方”进入独立设置页面。
- 设置内容中可选入物料，物料分组和物料类型来自基础信息。
- 原料池行可维护用量设置，38 管、罩壳、下杆等物料可按不同面料设置不同扣减和公式。
- 业务配置项可维护配置项、选项值、选项物料和限制条件。
- 业务配置项可配置显示条件，例如选择“系统 = 电机”后才出现“电机型号”和“遥控型号”。
- 保存设置后物料行数刷新，校验状态回到未校验。
- 未设置原料池或业务配置项不能提交审核。
- 未校验通过不能提交审核。
- 审核通过后生成 `V1`，版本快照包含主表、原料池、条件用量规则、业务配置项、选项物料和限制条件。
- 生效配方可停用，停用后不展示重新启用。
- 变更记录能看到新增、修改、保存设置、校验、提交审核、审核通过、驳回、停用和删除草稿。
