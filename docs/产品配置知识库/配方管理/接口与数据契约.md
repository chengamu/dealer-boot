配方管理拆成五个职责入口：配方档案、配方原料、配方选项、配方模拟、配方审核。配方档案只维护主信息、状态入口和提交审核；原料、选项、模拟分别维护草稿配置和校验；审核只查看冻结快照并执行通过或驳回。

## 配置聚合对象

配方原料、配方选项和配方模拟接口共享同一份草稿聚合数据；前端入口按职责拆成独立页面，原料和选项复用同一个轻量编排组件读取和保存自己的部分：

| 字段 | 说明 |
| --- | --- |
| `formula` | 配方主表展示信息，保存时可不传。 |
| `materials` | 配方原料，来源物料管理。 |
| `options` | 业务配置项定义。 |
| `optionValues` | 配置项可选值。 |
| `optionMaterials` | 选项值关联到原料池物料。 |
| `restrictions` | 异常限制，只处理禁用或提醒。 |
| `usageRules` | 用量规则，按原料池物料维护固定值或公式规则。 |

## 状态值

| 值 | 中文 | 说明 |
| --- | --- | --- |
| `DRAFT` | 草稿 | 可编辑、可删除、可维护原料、选项和模拟。 |
| `PENDING_REVIEW` | 待审核 | 锁定普通编辑和草稿配置，只能在配方审核中通过或驳回。 |
| `REJECTED` | 驳回 | 可编辑、可维护原料、选项和模拟，重新校验后再提交。 |
| `EFFECTIVE` | 生效 | 不可普通编辑，不可删除，可停用。 |
| `STOPPED` | 停用 | 不可编辑、不可删除、不可重新启用。 |

## 校验规则

- `formula_code` 在同租户未删除数据中唯一，由 Service 层校验。
- `formula_name + category_id + product_type_code + max_width_inch + max_height_inch` 在同租户未删除数据中唯一，由 Service 层校验。
- 产品分类必须存在且启用。
- 产品类型必须来自启用的 `product_type` 产品字典项。
- 配方原料、配方选项、配方模拟只能在草稿或驳回状态保存。
- 原料池只能选择启用物料，且同一配方不能重复选择同一物料。
- 原料池行必须有单位和有效用量规则；有条件用量规则时优先按条件规则校验，无条件规则时使用原料池行的旧单行用量字段兜底。
- 条件用量规则必须引用当前配方原料池中的物料。
- 条件用量规则引用的配置项和值必须存在于当前配方设置。
- 同一个物料下，同一个配置项值只能有一条启用条件规则。
- 每个存在条件规则的物料必须有一条默认规则，避免订单选择未命中时没有兜底。
- 固定用量必须填写固定数量；公式类口径必须填写公式。
- 公式变量第一版固定为 `orderWidthMm`、`orderHeightMm`、`widthDeductMm`、`heightDeductMm`、`lossRate`、`fixedUsageQty`。
- 配置项编码在同一配方下唯一。
- 配置项按 `sort_order` 升序返回和消费；设置页上移/下移后重排为 `10、20、30...`。
- `visibility_mode = ALWAYS` 时清空显示条件字段，订单端初始展示。
- `visibility_mode = CONDITIONAL` 时必须选择另一个配置项和该配置项下的选项值；条件配置项不能选择自己。
- 条件显示只控制订单端是否展示配置项。例如 `电机型号`、`遥控型号` 可设置为 `系统 = 电机` 时显示；当系统选择拉珠或无拉时不展示。
- 上游选择变化导致下游配置项隐藏时，下游已选值应由下单端清空，且其关联物料不再带出。
- 选项值在同一配置项下唯一。
- 选项物料必须来自当前配方原料池。
- 限制条件只能引用当前配方存在的配置项和值。
- 保存配方原料或配方选项后，主表 `material_line_count / configured_flag` 自动刷新，对应校验状态回到 `NOT_VALIDATED`。
- 提交审核前必须同时满足原料校验、选项校验和模拟校验为 `PASS`。
- 提交审核时生成待审核快照；审核通过时把该快照固化为正式 `pc_formula_version` 版本并更新当前版本字段。
- 停用后不允许重新启用；后续继续使用时应重新创建草稿版本审核。
- `pc_formula_version` 负责冻结审核内容，`pc_change_log` 负责记录操作流水，两者不能互相替代。

## AI 代办契约

| 项目 | 内容 |
| --- | --- |
| 支持意图 | 新增配方、修改草稿或驳回配方、维护配方原料、维护用量规则、维护配方选项、维护选项物料、维护异常限制、运行配方模拟、提交审核、审核通过、驳回、停用、查询变更记录。 |
| 必填槽位 | 配方编号、配方名称、产品分类、产品类型、最大宽度、最大高度；原料池行需要物料和单位；条件用量规则需要物料、计量口径、固定用量或公式；配置项需要编码和名称。 |
| 可推断字段 | 状态默认草稿；尺寸摘要由后端生成；保存设置后配置状态由后端刷新；版本号审核通过时生成。 |
| 必须追问 | 缺少产品分类、产品类型、最大尺寸、原料池物料、单位、用量规则、默认规则、配置项编码或名称时必须追问。 |
| 保存前确认 | 展示配方编号、名称、分类、产品类型、最大宽高、原料行数、配置项数量、限制条件数量、原料校验、选项校验和模拟校验状态。 |
| 禁止暴露 | 不要求用户理解数据库字段名；不承诺成本试算、打样记录和订单下单能力。 |
