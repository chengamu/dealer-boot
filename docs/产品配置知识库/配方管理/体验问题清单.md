# 配方管理体验问题清单

本文只记录当前仍需回归确认的体验问题。已固化为业务规则、接口契约或已确认修复的问题，直接移出本清单。

## 配方原料

- 需要用真实物料数据回归物料选择弹窗：属性分组、物料类型、物料编码、物料名称过滤是否够顺手，是否默认只选启用物料。
- 需要用真实用量规则回归用量设置弹窗：横向滚动、规则摘要和公式输入是否影响技术科连续录入。

## 条件与公式

- 需要用真实“面料 + 副面料 + 罩壳”数据回归选项物料属性表达式，例如 `面料.厚度 + 副面料.厚度 > 12`、`罩壳.重量 > 2.5`。
- 需要回归配置项或选项值被异常限制、用量规则、内部变量取值规则、关联物料引用时，删除失败原因是否能明确指向引用位置。

## 业务配置项

- 需要用真实多级选项树回归长名称、上移下移、新增下级和删除引用保护是否顺手。
- 需要用真实下单分支回归“系统 = 电机后展示电机型号、遥控型号”这类正常分支配置是否清楚。

## 国际化与展示

- 需要中英文环境各回归一次，确认页面不出现原始 i18n key，数量、用量和日期时间展示符合规则。

## 校验与提交流程

- 需要完整跑一次“配方原料 -> 业务配置项 -> 模拟 -> 校验 -> 提交审核 -> 审核通过”的回归，确认失败原因可稳定查看，版本快照可回看。
