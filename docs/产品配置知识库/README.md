# 产品配置知识库

本文是产品配置相关稳定知识库入口。当前索引基础信息、配方管理、可售产品、邮费模板和价格设置；其它业务域确认重写前不预铺目录。

价格设置的下游入口见 [商家体系 / 订单测算与客户报价](../商家体系/客户管理/报价管理/接口与数据契约.md)。订单测算只使用已就绪的可售产品、冻结配方版本、价格设置和启用邮费模板；确认报价冻结结果，转销售订单时不重新调用产品配置和价格引擎。

通用文档治理、人工测试闭环、可检索性与引用关系规则见 [文档治理规则](../项目配置和代码风格/documentation-governance.md)。AI 代办通用规则见 [AI代办能力规范](AI代办能力规范.md)。

## 文档形态

- `操作手册.md`：当前正确用法、录入顺序、业务说明和业务化 AI 说明。
- `接口与数据契约.md`：页面路由、接口、权限、字段、状态、选择器、可检索性、AI 工具契约。
- `体验问题清单.md`：当前仍需处理或明确暂不处理的体验问题；修复确认后直接移除。

## 适用范围

- 基础信息
- 配方管理：配方档案、配方原料、业务配置项、配方模拟、配方审核
- 产品定价：可售产品、邮费模板、价格设置

其它产品业务域确认后再创建目录，不预铺空目录。
