# 05 - 价格、发布与订单快照模块

## 1. 模块目标

这个模块解决：

```text
怎么报价
发布前能不能卖
发布后订单读取哪个版本
订单如何冻结配置、价格、工程配置、资料快照
```

前置条件：

```text
基础资料可引用
工程配置可预览和检查
产品配置可求值
```

价格中心不负责工程构成维护，也不负责生产用量和工艺扣减。当前阶段发布前只消费工程配置的预览和检查结果；真实生产用量、裁切扣减和生产 BOM 后续由 ERP / 生产工艺模块负责。

## 7. 一期验收

```text
价格能算出命中明细
发布检查能阻止缺配置、缺价格、缺资料、缺工程配置的产品上线
发布包不可变
订单保存的是快照，不是实时引用草稿
销售只读页只展示发布版本
```
