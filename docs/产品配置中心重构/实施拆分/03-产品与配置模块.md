# 03 - 产品与配置模块

## 1. 模块目标

产品与配置模块解决：

```text
客户买什么
客户下单时看到什么
客户能选什么
答案来自字典、物料还是组件包
选择答案后触发哪些规则
```

不解决：

```text
最终价格
最终生产 BOM
库存和采购
工程尺寸能力
生产尺寸扣减
用量公式
```

前置条件：

```text
基础资料已能提供物料、面料、组件包和资料资产
工程配置已能维护工程方案、构成项、可选范围、能力规则和带出规则
销售产品绑定工程方案版本后，才能进入发布检查和订单下单
```

## 2. 页面

```text
销售产品
标品 SKU
配置模板
问题组模板
配置问题
配置答案
配置求值器
```


## 4. 后端建议

Controller：

```text
ProductConfigController
```

Service：

```text
ProductConfigService
```

Engine：

```text
ConfigEvaluationEngine
```

`ConfigEvaluationEngine` 要从现在的必填校验升级为：

```text
返回可见问题
返回可选答案
返回禁用原因
返回必填校验
返回答案带出的组件
返回答案绑定的资料
返回工程配置预览结果
```

边界：

```text
配置规则只处理客户可见性、必填、默认值、轻量禁用提示
生产尺寸、裁切扣减、真实生产用量属于 ERP / 生产工艺，不在产品配置阶段处理
答案不直接承载工程构成逻辑；答案应映射工程构成项、工程可选项或工程带出规则入口
```


## 6. 必须尽早做试算器

配置求值器不能后置太久。

最小输入：

```text
templateVersionId
width
height
selectedOptions
```

最小输出：

```text
visibleQuestions
availableOptions
disabledOptions
validations
autoComponents
mediaAssets
warnings
blockers
```

## 7. 一期验收

```text
能创建一个销售产品
销售产品能绑定工程方案版本
能绑定一个配置模板
能维护问题和答案
答案能引用字典、物料或组件
输入已选答案后，后端返回可见问题、可选答案、校验结果、工程配置预览和缺失提示
```
