## 3. 一期主线

一期只先打通一条产品线，例如卷帘或斑马帘：

```text
基础资料
  -> 统一物料资料 / 物料属性 / 单位 / 组件包 / 资料资产
  -> 工程配置
  -> 工程方案 / 构成项 / 可选范围 / 能力规则 / 带出规则
  -> 工程配置预览 / 工程配置检查
```

业务依赖顺序必须保持：

```text
基础资料
  -> 工程配置
  -> 产品配置中心消费工程配置
```

说明：

```text
基础资料提供统一物料主数据、物料属性、单位、组件包和资料资产，是未来 ERP、工程配置、采购和库存共同消费的主数据底座。
工程配置基于基础资料定义产品系列由哪些构成项组成、每个构成项可选什么、什么条件下能选或不能选、应该带出什么组件包或资料。
产品配置中心先保留，不继续扩展；后续只作为客户下单问题和答案的展示层，消费工程配置。
```

---

## 4. 第一批建议

第一批不要铺满所有页面，先做：

```text
1. 基础资料：统一物料资料、物料属性、单位、组件包、资料资产
2. 基础资料：补物料规格摘要 spec_summary、采购/库存/使用单位等 ERP 主数据字段
3. 工程配置：工程方案、工程方案版本、工程构成项、构成项可选范围
4. 工程配置：工程能力规则、工程带出规则、标品固定工程配置
5. 工程配置：工程配置预览、工程配置检查
```

每批完成后再继续下一批。
